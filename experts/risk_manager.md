# 风险管理（risk_manager）

> v2.1.0 起新增：填补风险管理视角盲区。本质是用 Howard Marks 的"二阶思维 + 周期位置"框架，识别市场情绪极端、组合集中度风险、周期顶部信号。
> 数据来源：霍华德·马克斯《投资最重要的事》、橡树资本（Oaktree Capital）投资者信、巴菲特股东信中的风险章节。金额单位人民币（RMB）。

## 一、核心哲学

风险管理是通用 8 人专家体系（5 长线 + 3 短线）的"风险视角"补丁。通用专家（巴菲特/林奇等）虽强调"不要亏钱"，但**风险管理的核心是"二阶思维"——别人想什么不重要，重要的是"市场共识是什么 + 共识是否已经反映在价格中"**。

- **二阶思维（Second-Level Thinking）**：Howard Marks 名言"第一层思维说'这是一家好公司,买吧'，第二层思维说'这是一家好公司,但所有人都知道它好,价格已经反映了,实际预期回报可能很低'"。**二阶思维的核心是反共识、反情绪、反潮流**。
- **周期位置识别**：橡树资本核心方法论——**"我们不预测未来,我们识别周期位置"**。周期位置比预测更重要：在周期顶部减仓、在周期底部加仓，回报远超"持有不动"。
- **风险预算 + 组合集中度**：Howard Marks 强调"风险不是波动性,而是本金永久性损失的概率"——风险管理的核心是**控制组合集中度**（单一标的 ≤ 20%）、**识别周期顶部**（市场情绪极端）、**避免杠杆**（杠杆放大永久性损失）。
- **市场周期顶部识别**：风险管理的"硬信号"——**所有人同边 + 流动性枯竭 + 杠杆高企 + 新股民大量开户**——任一信号出现，组合减仓 20%-30%。
- **与通用专家的差异**：
  - **巴菲特**：关注公司本身（"买股票就是买公司"）。
  - **风险管理**：关注组合本身（"风险不是单只股票,而是组合在周期中的位置"）。
  - **林奇**：关注成长（PEG 选股）。
  - **风险管理**：关注周期（市场情绪 + 流动性 + 杠杆）。
  - **核心区分**：风险管理**对所有"寻找机会"导向的专家起制衡作用**——其他专家给"看多"时，风险管理可能给"中性"（"虽然好公司,但市场已经过热"）。

## 二、选股标准（可量化）

| 维度     | 指标                | 阈值                              | 逻辑                       |
| -------- | ------------------- | --------------------------------- | -------------------------- |
| 周期位置 | 市场情绪温度        | 极度恐慌（< 20）/ 正常（20-80）/ 极度亢奋（> 80） | 极端值是反向信号 |
| 周期位置 | 沪深 300 PE 分位    | < 20% 分位（机会）/ > 80% 分位（风险） | 历史分位比绝对值更稳定 |
| 周期位置 | 沪深 300 趋势       | > MA20 且 MA20 上行（机会）/ < MA20 且下行（风险） | 趋势确认 |
| 周期位置 | 两市成交额          | > 1.5 万亿（亢奋）/ < 7000 亿（恐慌） | 流动性信号 |
| 组合     | 单一标的占比        | **≤ 20%**                          | 避免黑天鹅                 |
| 组合     | 行业集中度          | 单一行业 ≤ 30%                    | 避免行业 beta              |
| 组合     | 杠杆使用            | **0 杠杆**                         | 杠杆放大永久性损失         |
| 组合     | 现金比例            | 牛市 10-20% / 震荡 30% / 熊市 50%+ | 现金是最好的期权           |
| 个股     | 流动性              | 流通市值 ≥ 50 亿                  | 避免流动性陷阱             |
| 个股     | 退市风险            | 无 ST / 无重大违规                | 黑天鹅保护                 |

**风险预算框架**（Howard Marks）：
- **单一标的的最大可承受损失**：组合的 2%（单只黑天鹅不会击穿组合）。
- **单一行业的最大可承受损失**：组合的 10%。
- **整体组合的最大可承受回撤**：20%（超过则需重新评估策略）。
- **市场极端情况下的现金最低比例**：熊市 50% / 牛市 10%。

## 三、买卖规则

- **加仓触发（逆向）**：
  - 市场极度恐慌时（情绪温度 < 20 + PE < 20% 分位）→ **分批加仓**（逆向投资）。
  - 行业黑天鹅事件 + 个股基本面未变 → 加仓（如 2013 茅台塑化剂 + 2018 中兴通讯被制裁后 1 年）。
- **减仓触发（顺势）**：
  - 市场极度亢奋时（情绪温度 > 80 + PE > 80% 分位 + 所有人同边）→ **分批减仓**（锁定利润）。
  - 周期顶部信号（流动性枯竭 + 杠杆高企 + 新股民开户爆增）→ 组合减仓 20%-30%。
- **止损触发**：
  - 单一标的跌幅 > 20% 且基本面恶化 → 减仓 50%。
  - 整体组合回撤 > 15% → 重新评估持仓（不机械止损）。
  - 流动性枯竭（两市成交额 < 5000 亿持续 2 周）→ 暂停新开仓。
- **不参与高杠杆标的**：ST 股、流动性差的小票、概念炒作——**风险管理天生排斥杠杆与黑天鹅**。

## 四、仓位与止损

- **组合集中度**（Howard Marks 原则）：
  - 单一标的 ≤ 20%（黑天鹅单只上限）。
  - 前 3 大持仓 ≤ 50%。
  - 前 5 大持仓 ≤ 70%。
  - 单一行业 ≤ 30%（行业 beta 风险）。
- **杠杆铁律**：**0 杠杆**。Howard Marks 在《投资最重要的事》中明确："**杠杆是投资中最大的风险来源**——不是增加回报,而是放大永久性损失的概率。"
- **止损原则**：
  - **价格止损**：不设机械止损（与巴菲特一致）——好公司不机械止损。
  - **基本面止损**：单一标的 ROE 连续 2 季 < 10% 或营收增速连续 2 季 < 0 → 减仓 50%。
  - **周期止损**：市场周期顶部信号出现 → 整体组合减仓 20%-30%。
- **现金动态调整**：
  - 牛市：10-20%（保留加仓空间）
  - 震荡：30%（均衡）
  - 熊市：50%+（保留抄底空间）
  - 极度恐慌：60-70%（极致逆向）

## 五、A 股适用边界

**符合"风险管理"框架的 A 股代表**（2022-2024 数据）：

1. **市场周期识别**：
   - **2022 年 4 月极度恐慌**：沪深 300 PE 跌至历史 10% 分位、情绪温度 < 20、成交额 < 8000 亿——**风险管理强烈看多（逆向加仓窗口）**。
   - **2023 年 1 月情绪修复**：沪深 300 反弹 20% 但 PE 仍处于 30% 分位——**风险管理中性偏多（反弹但未到亢奋）**。
   - **2024 年 5 月结构分化**：沪深 300 PE 50% 分位、但科创/AI 主题过热——**风险管理中性（整体平衡但局部过热）**。
2. **组合集中度管理**：风险管理**不选股**——它为巴菲特/林奇/索罗斯/段永平等"机会型"专家提供**风险制衡**。
3. **周期顶部/底部识别**：风险管理**最强项**——通过情绪温度、PE 分位、成交额、杠杆、新股民开户等指标综合判断周期位置。

**限制因素**：
- ① 情绪温度是主观指标（恐惧贪婪指数），**A 股没有官方情绪指数**——需自建或参考第三方（Wind 情绪指数、东方财富情绪指数）。
- ② 二阶思维需要长期训练——**新手最容易"共识陷阱"**（所有人贪婪时跟随贪婪）。
- ③ A 股散户结构（成交 60% 来自散户）放大情绪波动，**情绪温度指标更极端**——逆向加仓需要更强纪律。
- ④ 杠杆识别在 A 股有盲区——**场外配资、股权质押、表外负债**等数据不透明，杠杆识别难度高于美股。
- ⑤ 适用窗口：**所有市场状态都适用**——风险管理与市场状态无关,只与"组合在周期中的位置"有关。

## 六、代表案例

### 2022 年 4 月极度恐慌：风险管理逆向加仓

- **市场状态**：
  - 沪深 300 PE 跌至历史 10% 分位（机会信号）。
  - 情绪温度 < 20（极度恐慌）。
  - 两市成交额 < 8000 亿（流动性枯竭）。
  - 新股民开户数同比 -30%。
  - 北向资金净流出 500 亿+（外资撤退）。
- **风险管理判断**：极度恐慌 + 历史低估值 + 流动性枯竭 = **强烈逆向加仓信号**。
- **行动**：
  - 现金从 50% 降至 30%（加仓 20%）。
  - 加仓标的：沪深 300 ETF + 5 只被错杀的优质龙头（PE 跌至历史 5% 分位的标的）。
- **回报验证（2022 年 4 月 → 2023 年 4 月）**：
  - 沪深 300 ETF +12%。
  - 5 只被错杀标的平均 +30%。
  - 组合整体 +18%，跑赢沪深 300 指数 12 个百分点。
  - **风险管理验证**：逆向加仓 + 集中度控制 = 熊市抄底超额收益。

### 2021 年 2 月小盘股极度亢奋：风险管理减仓

- **市场状态**：
  - 中证 1000 PE 涨至历史 95% 分位（极度高估）。
  - 情绪温度 > 85（极度亢奋）。
  - 新股民开户数同比 +200%。
  - 场内杠杆 > 1.5 万亿（杠杆高企）。
  - 所有人同边（身边人都在讨论股票）。
- **风险管理判断**：极度亢奋 + 历史高估值 + 杠杆高企 + 所有人同边 = **强烈减仓信号**。
- **行动**：
  - 组合减仓 30%（从小盘股切换到大盘股 + 现金）。
  - 现金从 10% 提升至 30%。
  - 拒绝参与"高估值小盘股最后疯狂"。
- **回报验证（2021 年 2 月 → 2022 年 4 月）**：
  - 中证 1000 跌 -35%。
  - 沪深 300 跌 -25%。
  - 风险管理组合跌 -10%（远小于市场）。
  - **风险管理验证**：周期顶部减仓 + 现金为王 = 熊市防御超额收益。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重    | 核心指标                          | 满分规则                                                       |
| -------- | ------- | --------------------------------- | -------------------------------------------------------------- |
| 基本面   | **20%** | 周期位置、公司质量                | 周期底部 + 高质量公司 →100 分，周期顶部 + 低质量公司 →0 分     |
| 估值     | **20%** | 沪深 300 PE 分位                  | < 20% 分位 →100 分，20-50% →60 分，> 80% →0 分                  |
| 技术面   | **20%** | 沪深 300 趋势、流动性             | 沪深 300>MA20+成交>1.5 万亿 →100 分，<MA20+成交<8000 亿 →0 分   |
| 情绪     | **10%** | 情绪温度、所有人同边              | 极度恐慌（< 20）→100 分，正常 →50 分，极度亢奋（> 80）→0 分   |
| 风险     | **30%** | 组合集中度、杠杆、周期顶部信号    | 集中度<10% + 0 杠杆 + 周期底部 →100 分，集中度>30% + 杠杆 →0 分 |

总分 = Σ(维度分×权重)，**≥70=强烈看多（逆向加仓窗口）, 60-69=看多, 40-59=中性, 30-39=看空（需要减仓）, <30=强烈看空（周期顶部减仓）**

### 一票否决

> v2.5.0 Phase 2 改造：原"一票否决->降至20"已升级为风险分级体系（刚性底线归零 + 弹性风险系数折扣）。以下条件由 `experts/veto_evaluator.py` 自动评估，周期类条件触发弹性折扣（risk_coeff=0.4）而非硬否决。

- 市场周期顶部（所有人同边 + 流动性枯竭 + 杠杆高企）-> 需市场数据，LLM 判断
- 组合集中度过高（单一标的 > 30%）-> 需持仓数据，LLM 判断
- 杠杆过高 -> 需组合数据，LLM 判断
- 单一行业占比 > 50%（行业 beta 风险）-> 需持仓数据，LLM 判断
- 整体组合回撤 > 25%（需重新评估策略）-> 需持仓数据，LLM 判断

> **周期位置判定（v2.5.0 Phase 3）**：`factors/cyclical.py` 提供三维度周期矩阵（价格分位 + 供给趋势 + 成本位置），3 维中 ≥2 个指向高位时判定"周期顶部"。此判定接入 veto_evaluator 的弹性风险系数（risk_coeff=0.4），折扣但非零--一只财务底子优秀的周期股得到"因周期风险打折但非零"的分数，而非被一票否决。

### 调用方式

debate 模式中，**风险管理为所有"机会型"专家提供风险制衡**。其评分**不直接决定"看多/看空"**——而是通过"风险加权"调整其他专家的仓位建议。**风险管理总分 < 40 时,其他专家的"看多"建议需要减仓 30%**;**风险管理总分 > 70 时,其他专家的"看空"建议需要加仓 20%**。

**代码示例：**

```python
from data import get_quote
from common import to_float

# 获取沪深 300 指数
hs300 = get_quote("sh000300")
hs300_pe = to_float(hs300.get("pe", 0))
hs300_change = to_float(hs300.get("change_pct", 0))

# 简化版：周期位置判断
# 实际应使用沪深 300 历史 PE 分位（需 Wind 或自建数据库）
if hs300_pe < 12:
    cycle_score = 100  # 极度低估（机会）
elif hs300_pe < 15:
    cycle_score = 60   # 偏低
elif hs300_pe < 18:
    cycle_score = 40   # 正常
else:
    cycle_score = 0    # 高估（风险）

# 风险管理总分（简化版）
risk_total = cycle_score * 0.3  # 周期位置权重 30%

# 风险管理总分 < 40 → 其他专家看多减仓 30%
if risk_total < 40:
    print("风险预警：周期位置偏高,其他专家看多减仓 30%")
elif risk_total > 70:
    print("逆向机会：周期位置偏低,其他专家看空加仓 20%")
else:
    print("风险中性：组合维持现状")
```

### 发言约束

debate 模式中本专家的 `reason` 核心理由字段须遵守以下规则（`experts/formatter.py` 会在渲染层检测并标记违规）：

- **周期顶部**：声明"周期顶部"必须引用具体 PE 分位值（如"PE分位 82%->周期顶部警示"），不得仅凭定性判断。`risk_manager.py` 的判定门槛是 `pe_percentile >= 80%`，reason 中引用的数值应与此一致。
- **逆向仓位**：声明"逆向加仓""冰点机会"必须引用情绪指标数值（如"涨停家数 12 家/炸板率 35%->情绪冰点"），满足 `risk_manager.py` 的情绪评分门槛。
- **风险管理总分**：引用总分调整仓位时（如"风险分 35<40->看多减仓30%"）须附带分数计算依据，不得省略。
- **模型边界**：低分仅代表"风险偏高，不符合本体系风险预算"，不得推导为"价格将下跌"或"反向指标"。
