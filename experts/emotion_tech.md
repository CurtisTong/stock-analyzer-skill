# 情绪技术复合（emotion_tech）

v2.1.0 起合并：炒股养家（情绪流）+ 作手新一（强势股低吸）。

## 核心视角

- 涨停家数 + 情绪周期阶段（养家维度）
- K 线反转 + 缩量 + 支撑位（作手新一维度）
- 情绪权重 46.5%（本专家最高）

## 评分矩阵

| 维度 | 权重 |
|---|---|
| 基本面 | 6.5% |
| 估值 | 12% |
| 技术面 | 24% |
| 情绪 | 46.5% |
| 风险 | 11% |

## 一票否决

- 情绪周期处于退潮期（连续亏钱效应）
- 全市场跌停家数 > 50
- 非强势股（无涨停或连板历史）
- 调整中放量跌破前低

## 适用场景

- 日/周持仓周期
- 情绪转折首板 / 强势股回调低吸
- 冰点机会识别（参养家冰点逻辑）
