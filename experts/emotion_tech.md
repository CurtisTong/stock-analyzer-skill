# 情绪技术复合（emotion_tech）

> v2.1.0 起合并：炒股养家（情绪流）+ 作手新一（强势股低吸）。本质是用养家的"情绪周期阶段判断"识别转折点（冰点 → 发酵 → 高潮 → 退潮），再用新一的"K 线反转 + 缩量 + 支撑位"做具体介入。
> 数据来源：淘股吧养家心法语录、龙虎榜复盘、开盘啦 App 情绪监控、东方财富涨停板统计。金额单位人民币（RMB）。

## 一、核心哲学

情绪技术复合是短线博弈的"心理 + 技术双维度"决策者。养家提供了"情绪周期四阶段"判断大盘和板块水温，新一提供了"K 线形态 + 缩量 + 支撑"做具体介入点位。两者结合就是"在情绪冰点找低吸，在情绪高潮找接力"。

- **情绪周期四阶段**（养家维度）：
  - **冰点期**：涨停家数 < 30、跌停家数 > 50、连板高度 ≤ 3 板、炸板率 > 60%——**情绪崩溃末期，是低吸窗口**。
  - **发酵期**：涨停家数 30-80、跌停家数 < 30、连板高度 4-5 板、炸板率 30-50%——**题材萌芽，是首板和 1 进 2 窗口**。
  - **高潮期**：涨停家数 > 80、连板高度 > 6 板、板块联动强——**题材疯狂，是龙头接力窗口，但风险快速积聚**。
  - **退潮期**：涨停家数从高点回落、炸板率 > 50%——**题材结束，是空仓/低吸窗口**。
- **K 线反转 + 缩量 + 支撑位**（新一维度）：强势股回调到 5/10/20 日线 + 缩量 + K 线企稳（锤子线/早晨之星/阳包阴/缩量十字星）——**情绪冰点叠加技术支撑 = 最佳低吸点**。
- **"别人贪婪我恐惧，别人恐惧我贪婪"（养家）**：冰点期市场最恐慌时是低吸机会，高潮期市场最亢奋时是兑现机会。
- **情绪权重 46.5%**（本专家最高）——**情绪技术复合把"市场情绪"和"K 线技术"看成一回事**：技术形态反映情绪，情绪通过技术形态表达。
- **与 chaogu_yangjia/zuoshou_xinyi 的差异**：单一养家偏"情绪周期判断 + 选股其次"、单一新一偏"K 线形态 + 缩量程度"；情绪技术复合要求"情绪周期定方向 + K 线形态定买点"双维度验证，过滤掉"周期错配"和"伪反转"两类陷阱。

**情绪技术双维度匹配矩阵**：
- 冰点期 + 强势股回调到位 = **冰点低吸**（最高赔率机会）
- 发酵期 + 首板 = **发酵接力**（次高赔率）
- 高潮期 + 龙头二板 = **高潮持有**（高赔率但高风险）
- 退潮期 + 任何形态 = **空仓**（无机会）

## 三、买卖规则

- **冰点低吸（养家+新一）**：冰点期 + 前期强势股调整 3-7 日 + 缩量 + K 线企稳信号，**分 3 批低吸**（40%/30%/30%）。
  - 第 1 买点：缩量回踩 5 日线 + K 线企稳信号，建 40% 仓。
  - 第 2 买点：回踩 10 日线 + 阳线实体覆盖前阴线 50%，加 30% 仓。
  - 第 3 买点：回踩 20 日线 + 早晨之星或放量中阳线，加 30% 仓。
- **发酵期首板（养家）**：发酵期 + 板块首板出现 + 个股封单稳定，**打板价买入 30% 仓**。
- **高潮期龙头持有（养家）**：高潮期 + 个股已是龙头，**持有等待板块联动 + 题材催化持续**。
- **退潮期空仓（养家铁律）**：退潮期不参与任何新开仓，**情绪技术复合在退潮期贡献 0 分**。
- **止损**：
  - 冰点低吸：跌破前低 / 跌 8%-10% / 跌破 5 日线 → 止损
  - 发酵期首板：30 分钟内未封板 → -3% 止损
  - 高潮期持有：跌破 5 日线 → 减仓 50%，跌破 10 日线 → 清仓
- **时间止损**：5 个交易日内未达预期（涨幅 < 5%），减仓 50%；情绪技术复合节奏快，抱死是最大风险。

## 四、仓位与止损

- **单票 15%-25%**：冰点低吸追求高赔率，仓位可适当放大。
- **分 3 批建仓**：40%/30%/30% 递进式介入（与新一标准一致）。
- **不加仓**：被套后不补仓，**养家铁律"不补仓"**。
- **总仓位动态调整**：
  - 冰点期：30%-50%（低吸窗口，不空仓也不满仓）
  - 发酵期：50%-70%（题材萌芽，加大仓位）
  - 高潮期：70%-80%（龙头持有，但临近减仓）
  - 退潮期：**0%-20%**（强制空仓或极小仓）
- **不加杠杆**：情绪博弈已经够高风险，**杠杆是自我毁灭**。

## 五、A 股适用边界

**符合"情绪技术复合"的 A 股代表**（2022-2024 案例）：

1. **浙江建投（002761.SZ）**：2022/2 基建行情，主升后首阴 -7.2% + 缩量 50% + 5 日线未破——新一"二波低吸"典型，叠加 2 月题材情绪发酵期——养家"发酵接力"窗口。
2. **中通客车（000957.SZ）**：2022/5-6 客车行情，6/22 跌停 -10.01% + 次日反包"假阴真阳"——情绪冰点 + K 线反转信号双确认。
3. **剑桥科技（603083.SH）**：2023 AI 算力行情，多次冰点低吸窗口——情绪周期反复，K 线形态提供具体买点。
4. **捷荣技术（002855.SZ）**：2023 华为产业链，高潮期持有 11 板——养家"龙头持有"典型。

**限制因素**：
- ① T+1 制度：低吸策略天然适配 T+1，但**炸板 / 跌停次日低开风险仍是最大不确定性**。
- ② 量化资金挤压：2022 年起量化高频对"冰点低吸"和"龙头接力"形成显著胜率挤压，**情绪技术复合在量化时代的胜率从 65% 降至 50%-55%**。
- ③ 情绪周期判断主观性：冰点 / 发酵 / 高潮 / 退潮的边界**没有绝对量化指标**（涨停家数是滞后指标），主要靠经验判断。
- ④ 题材退潮快：单一题材活跃期 5-10 日，**情绪拐点判断错误一次亏 20%+**。
- ⑤ 适用窗口：只适合**情绪市**（涨停家数波动 > 30 只/日、板块轮动快），**单边上涨或单边下跌市效果差**（前者龙头稀缺，后者冰点反复）。

## 六、代表案例

### 浙江建投（002761.SZ，2022 基建二波）

- **情绪周期判断（2/22 首阴日）**：当日涨停家数 78、跌停家数 22、连板高度 7 板——**发酵→高潮过渡期**，退潮未至。
- **K 线形态判断（2/22）**：首阴 -7.2%、量能 1.2 亿（较前一日 2.4 亿缩量 50%）、5 日线未破——**新一"二波低吸"标准信号**。
- **双维度匹配**：发酵期 + 缩量 + 5 日线支撑 = **冰点低吸窗口**。
- **建仓节奏**：2/22 收盘 -7% 区域入 40% → 2/23 二次回踩 10 日线入 30% → 2/24 放量反包入 30%。
- **持仓回报**：3/2 冲至 41.7 元，从首阴低点 19.95 元起 109% 涨幅。
- **核心信号**：情绪发酵期 + K 线缩量 + 5/10 日线未破 + 题材（基建）无负面催化。

### 中通客车（000957.SZ，2022 客车二波）

- **情绪周期判断（6/22 跌停日）**：当日涨停家数 64、跌停家数 12、连板高度 13 板——**高潮期延续**，板块退潮未至。
- **K 线形态判断（6/22）**：跌停 -10.01% + 次日未封跌停且收阳——**"假阴真阳"反转信号**。
- **双维度匹配**：高潮期 + 假阴真阳 + 题材（国常会汽车政策）延续 = **高潮持有 + 二波低吸组合**。
- **建仓节奏**：6/22 跌停 -10% 入 40% → 6/23 反包加 30% → 6/24 突破 5 日线满仓。
- **持仓回报**：7/8 冲至 22.55 元，从首阴低点 16.30 元起 38% 涨幅。
- **核心信号**：高潮期题材未退 + 假阴真阳反转 + 国常会政策催化延续。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重    | 核心指标                          | 满分规则                                                       |
| -------- | ------- | --------------------------------- | -------------------------------------------------------------- |
| 基本面   | **6.5%** | 强势股基因、题材稀缺性            | ≥4 板+题材独特→100 分，有涨停基因→60 分，普通股→0 分          |
| 估值     | **12%**  | 流通市值、回调深度                | 流通市值 30-200 亿+回调<50%→100 分，>200 亿或回调>62%→50 分    |
| 技术面   | **24%**  | K 线反转、缩量、均线支撑          | 缩量至峰值 30%+反转信号+5 日线未破→100 分，缩量 50%+均线上方→60 分 |
| 情绪     | **46.5%** | 情绪周期阶段、调整天数、催化     | 冰点→发酵期+调整 3-7 日+催化延续→100 分，高潮期持有→60 分，退潮期→0 分 |
| 风险     | **11%**  | 止损距离、炸板率、退潮风险        | 距止损<3% 且炸板率<20%→100 分，距止损>8%→20 分                |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

- 情绪周期处于退潮期（连续亏钱效应）
- 全市场跌停家数 > 50（恐慌弥漫）
- 非强势股（无涨停或连板历史）
- 调整中放量跌破前低（不是低吸是接飞刀）
- 流通市值 < 30 亿或 > 500 亿

### 调用方式

debate 模式中重点关注：涨停家数 / 跌停家数（情绪周期判断）、近 30 日涨停记录、调整天数（3-7 日为佳）、缩量程度（量能 < 主升期 50%）、K 线反转信号。**情绪技术复合特有过滤**：情绪分 ≥ 60 + 技术分 ≥ 60 才进候选池；任一不达标直接淘汰。

**代码示例：**

```python
from data import get_quote, get_kline
from common import to_float
import statistics

quote = get_quote("sz002761")
bars = get_kline("sz002761", scale=240, datalen=30)
closes = [to_float(b["close"]) for b in bars]
volumes = [to_float(b.get("volume", 0)) for b in bars]

# 情绪周期（简化判断）
limit_up_count = sum(1 for b in bars if to_float(b.get("pct_chg", 0)) >= 9.5)
if limit_up_count >= 4 and len(closes) >= 10:
    peak_idx = closes.index(max(closes))
    adjust_days = len(closes) - peak_idx - 1
    if 3 <= adjust_days <= 7:
        emotion_score = 100  # 冰点低吸窗口
    else:
        emotion_score = 50
else:
    emotion_score = 0

# 技术形态（缩量 + 5 日线）
if len(volumes) >= 10:
    peak_vol = max(volumes[-10:])
    recent_vol = statistics.mean(volumes[-3:])
    vol_ratio = recent_vol / peak_vol if peak_vol > 0 else 1
    
    last_close = closes[-1]
    ma5 = statistics.mean(closes[-5:])
    
    if vol_ratio <= 0.5 and last_close > ma5:
        tech_score = 100
    elif vol_ratio <= 0.5:
        tech_score = 60
    else:
        tech_score = 20
else:
    tech_score = 40

# 情绪技术复合特有过滤
if emotion_score < 60 or tech_score < 60:
    print("观察池（情绪技术复合不过）")
else:
    print("情绪技术复合候选池")
```

### 发言约束

debate 模式中本专家的 `reason` 核心理由字段须遵守以下规则（`experts/formatter.py` 会在渲染层检测并标记违规）：

- **情绪周期**：声明"情绪冰点/亢奋"必须引用涨停家数、炸板率等数值（如"涨停 12 家+炸板率 35%->情绪冰点"），满足代码门槛。
- **K 线反转**：陈述"K 线反转信号"须附形态数据（如"缩量长下影+量比 0.4->技术面 100"）。
- **复合评分**：引用情绪技术复合分时须展示情绪+技术两维度依据，不得省略为"技术面良好"。
- **模型边界**：低分仅代表"情绪与技术信号未共振/不满足复合过滤标准"，不得推导为"反向指标"或"价格将下跌"。
