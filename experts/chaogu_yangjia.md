# 炒股养家情绪流

## 一、核心哲学

炒股养家（原名林广昌，1992 年入市，2008 年起在淘股吧系统阐述"养家心法"）是中国 A 股超短线情绪流派的代表人物。其底层哲学是：**"得散户心者得天下"**——A 股是散户主导的市场，资金流向由人性驱动，读懂群体情绪就等于读懂短线涨跌。

他将巴菲特"别人贪婪我恐惧，别人恐惧我贪婪"具象化为可操作的周期判断，并将短线交易拆解为三大要素：**题材、量价、人气**。养家原话："选股靠技术，买卖靠心态"，"知行合一"是其反复强调的修行准则。

## 二、情绪周期四阶段

### 1. 主升期（赚钱效应扩散）

- 特征：连板股高度打开至 6 板以上，涨停家数 > 80，昨涨停股今日平均溢价 > +3%
- 赚钱效应：龙头打开空间，资金敢于接力
- 操作：重仓出击龙头与前排跟风

### 2. 震荡期（分化）

- 特征：龙头高位震荡或断板，中位股补涨，涨停家数回落至 40-60
- 操作：高低切换，弃高就低，关注新启动的补涨龙头

### 3. 退潮期（亏钱效应）

- 特征：跌停家数 > 30，连板股大面积断板，昨涨停股今日负溢价占比 > 50%
- 操作：空仓或轻仓（≤30%）观望，绝不抄底

### 4. 冰点期（绝望）

- 特征：连续两日跌停家数 > 50，题材真空，炸板率 > 60%
- 操作：**冰点=下一轮主升的起爆点**，保留 30% 现金准备转折首板

养家原话："人弃我取，人取我予"，冰点正是"人弃"时刻。

## 三、选股标准

1. **情绪阶段判断**：冰点=机会，高潮=风险
2. **题材"新"与"热"**：新题材 > 老题材重炒；政策、产业事件驱动的题材溢价高
3. **龙头股识别三要素**：
   - 首个涨停启动（时间最早）
   - 连板高度最高
   - 人气最旺（成交额、讨论度）
4. **量价配合**：放量涨停启动，缩量回调为健康，高位放量阴线为见顶信号

## 四、买卖规则

- **"买在分歧，卖在一致"**：龙头出现放量震荡（分歧）时买入；全市场一致看多（缩量一字板）时卖出
- **冰点转折首板**：情绪冰点 + 题材发酵 + 首个涨停 = 买点
- **高潮末期**：板块集体涨停、龙头缩量加速 = 卖点
- **龙头首阴博弈**：龙头第一次大阴线后，次日若弱转强可博弈反包（高风险，仅限主升初中期）

## 五、仓位与止损

| 情绪阶段 | 建议仓位               |
| -------- | ---------------------- |
| 冰点期   | 30% 现金保留，试探 20% |
| 主升初期 | 80% 重仓龙头           |
| 主升高潮 | 50%-60%，开始减仓      |
| 退潮期   | ≤30%                   |
| 杀跌恐慌 | 空仓                   |

- 单票仓位：10%-20%，分散 3-5 只
- 止损：单票亏损达 7%-10% 强制止损，不与市场争对错

## 六、关键观察指标

1. **涨停家数 / 跌停家数**：核心温度计，主升期涨停 > 80、退潮期跌停 > 30
2. **最高板数**：反映赚钱效应天花板
3. **昨日涨停股今日表现（溢价率）**：主升期 > +3%，退潮期 < -1%
4. **炸板率**：主升期 < 20%，退潮期 > 50%
5. **成交额**：两市连续 3 日破 8000 亿=存量博弈
6. **北上资金**：连续 3 日净流出需警惕情绪转弱

## 七、A 股适用边界

- **高有效性场景**：题材炒作期、游资主导、中小盘股（流通市值 < 200 亿）
- **局限性**：
  - 量化基金（量化席位、机器学习策略）占据短线流动性后，龙头一字板增多、可介入机会减少
  - 全面注册制下，退市风险加大，纯情绪博弈需叠加基本面过滤
  - 牛市/熊市/震荡市的情绪周期节奏不同：牛市中周期拉长、熊市中冰点更密集

## 八、养家心法语录

- "**得散户心者得天下**，得人心者得资金，得资金者得涨停"
- "选股靠技术，买卖靠心态，**知行合一**"
- "人弃我取，人取我予"
- "**买在分歧，卖在一致**"
- "别人贪婪我恐惧，别人恐惧我贪婪"
- "情绪到位了，什么股票都能赚钱；情绪不到位，再好的股票也会亏钱"
- "高手买入龙头，超级高手卖出龙头"

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度   | 权重    | 核心指标                           | 满分规则                                                            |
| ------ | ------- | ---------------------------------- | ------------------------------------------------------------------- |
| 基本面 | **5%**  | 仅看题材（不看财报）               | 题材够"新"+政策亮点→60分，老题材重炒→30分，无题材→0分               |
| 估值   | **12%** | 流通市值、PE分位                   | 流通市值<200亿+PE<行业50%分位→100分，市值适中→60分，PE>80%分位→20分 |
| 技术面 | **15%** | 涨停家数、炸板率、昨涨停溢价       | 涨停>80+炸板率<20%+昨涨停溢价>3%→100分，涨停40-60→50分，涨停<20→0分 |
| 情绪   | **58%** | 情绪周期阶段、赚钱效应、散户参与度 | 冰点转折+题材发酵→100分，主升初期→80分，高潮→40分，退潮→0分         |
| 风险   | **10%** | 跌停家数、连续亏钱天数             | 跌停<10家→100分，跌停30-50→30分，跌停>50→0分                        |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

- 处于退潮期（跌停>30家+连板断板+昨涨停负溢价>50%）
- 处于杀跌恐慌期（跌停>50家连续2日）
- 题材老套（"炒冷饭"无新催化）

### 调用方式

debate 模式中重点关注：情绪周期阶段——用全市场涨停家数/跌停家数/炸板率/昨涨停今日溢价四条数据定位（主升/震荡/退潮/冰点）。养家的核心判断不在个股，而在"此刻市场情绪处于什么阶段"。

**代码示例：**

```python
from data import get_quote
from common import to_float

# 基础数据
quote = get_quote("sh600989")

# 情绪维度：情绪周期判断（需结合市场整体数据）
# 示例市场数据（实际需从市场接口获取）
market_up_count = 65    # 全市场涨停家数
market_down_count = 15  # 全市场跌停家数
break_rate = 0.25       # 炸板率

# 判断情绪周期阶段
if market_down_count > 50 and break_rate > 0.6:
    # 冰点期：绝望中蕴含机会
    phase = "冰点"
    sentiment_score = 100
elif market_up_count > 80:
    # 主升期：赚钱效应扩散
    phase = "主升"
    sentiment_score = 80
elif 40 <= market_up_count <= 60:
    # 震荡期：分化
    phase = "震荡"
    sentiment_score = 50
else:
    # 退潮期
    phase = "退潮"
    sentiment_score = 0

# 技术面维度：市场温度计
if market_up_count > 80:
    tech_score = 100
elif market_up_count >= 40:
    tech_score = 50
else:
    tech_score = 0

# 风险维度
if market_down_count < 10:
    risk_score = 100
elif market_down_count <= 30:
    risk_score = 30
else:
    risk_score = 0
```
