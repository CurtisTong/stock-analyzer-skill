# 巴菲特投资哲学

> 数据截至 2024 年伯克希尔 13F 与年报；所有金额单位为美元（USD），A 股案例以人民币（RMB）计。

## 一、核心哲学

巴菲特（Warren Buffett）师从本杰明·格雷厄姆（Benjamin Graham），并经菲利普·费雪（Philip Fisher）与查理·芒格（Charlie Munger）改良，形成"以合理价格买入伟大生意并长期持有"的投资体系。

- **价值投资本质**：股票是公司所有权的凭证（"business ownership"），价格围绕**内在价值（intrinsic value）**波动，目标是寻找二者偏离的标的。
- **三要素**：`Great Business` + `Reasonable Price` + `Long Term`。芒格补充："以合理价格买入伟大公司，远胜于以伟大价格买入合理公司。"
- **复利**：1965-2023 年伯克希尔年化复合收益率 **19.8%**，同期标普 500 仅 9.9%，差异源自复利与时间。
- **市场先生（Mr. Market）**：源自 1987 年股东信，市场情绪化报价，应利用而非被其左右。

## 二、选股标准（可量化）

| 指标              | 阈值                             | 数据来源                    |
| ----------------- | -------------------------------- | --------------------------- |
| 长期 ROE          | **≥ 15%**（连续 10 年）          | 致股东信 1979、1991         |
| 毛利率            | **≥ 40%**（消费/制药）           | 1991 信"利润率最高"企业清单 |
| 净利率            | **≥ 15%**                        | 致股东信 2018               |
| 资产负债率        | **< 50%**（金融业除外）          | 致股东信多次强调            |
| 自由现金流（FCF） | **持续为正且占净利润 > 80%**     | 致股东信 2014               |
| 管理层            | 理性、坦诚、对股东友好、抵制惯性 | 1986、2014 信               |

**护城河四要素（economic moat）**：① 品牌（可口可乐）；② 成本（GEICO、BNSF 铁路）；③ 网络效应（运通卡）；④ 转换成本（苹果 iOS 生态、See's Candies 礼品卡）。巴菲特 1999 年《财富》文章首次系统化提出。

## 三、买卖规则

- **安全边际（Margin of Safety）**：买入价相对于内在价值折扣 **≥ 30%**（格雷厄姆原教旨，2014 信巴菲特仍坚持此比例）。
- **内在价值估算**：采用**两阶段 DCF**——未来 10 年自由现金流按增长率折现，第 10 年后永续增长率取 **3%**（美国名义 GDP 长期均值）。折现率取**美国 10 年期国债 + 风险溢价**。
- **加仓节奏**：当股价继续下跌且内在价值未受损时加仓（"be fearful when others are greedy"）；1991 年富国银行、巴菲特 3 次加仓至持仓 9%。
- **持有期**："Our favorite holding period is **forever**"（1988 致股东信）。但并非绝对，下设 4 个卖出条件。

## 四、仓位与止损

- **集中持仓**：2023 年末伯克希尔股票组合中，前 5 大持仓（苹果、美运、可口可乐、雪佛龙、卡夫亨氏）占股票市值约 **75%**；前 10 大占约 **85%**。
- **不止损哲学**：若公司基本面未恶化，价格下跌反而提供安全边际扩大机会；止损等于"割肉把好生意让给市场先生"。
- **四类卖出条件**（1996、2014 致股东信）：① 估值严重高估；② 找到更便宜的替代品；③ 公司基本面恶化（护城河被颠覆）；④ 收购整家公司（如 2010 BNSF 100% 收购、2016 苹果仓位由"二级市场买入"转为"长期持有"逻辑）。

## 五、A 股适用边界

**符合"巴菲特标准"的 A 股代表**（按 2023 年报数据）：

1. **贵州茅台（600519.SH）**：ROE 33%、毛利率 92%、净利率 53%、资产负债率 18%——全面超标；2013-2014 塑化剂危机 PE 跌至 9 倍，巴菲特原则下的典型买入窗口。
2. **片仔癀（600436.SH）**：ROE 24%、毛利率 79%、国家保密配方构成"品牌 + 牌照"护城河，2021 年股价回调 40% 视为合理买点。
3. **长江电力（600900.SH）**：ROE 15.5%、经营性现金流 ≈ 净利润 1.6 倍、合同模式（"准债券"）——典型"确定性收益"标的。

**限制因素**：① 信息披露透明度低于美股（关联交易、商誉减值调节空间大）；② 退市机制弱，"永久持有"承担政策风险；③ 散户结构（A股成交 60% 来自散户）放大波动，机会增多但需更强纪律。

**实操调整**：DCF 折现率建议取 **10%**（A 股波动率高于美股）；永续增长率取 **2-3%**（名义 GDP）；对国企需折扣 20% 治理风险。

## 六、代表案例

- **可口可乐（KO）**：1988 年首次建仓，至 1994 年累计投入 **13 亿美元**，2023 年末持仓市值 **约 250 亿美元**——35 年回报近 19 倍，期间从未减持，验证"永远持有"。
- **苹果（AAPL）**：2016 年 Q1 起建仓（经 Todd Combs 建议），2023 年末持仓 **1,742 亿美元**，占伯克希尔股票组合约 50%。关键判断：iPhone 已从"科技产品"转为"消费品"，护城河从"创新"转为"用户转换成本 + 服务生态"。
- **比亚迪（1211.HK）**：2008 年 9 月以 **18 亿港元**购入比亚迪 10% 股权，源自芒格一句"这家公司像早期丰田"。至 2022 年陆续减持前，最高浮盈约 **38 倍**——非典型巴菲特（重仓新兴市场 + 单一押注），反映芒格对伯克希尔风格的拓展。
- **See's Candies**：1972 年以 **2,500 万美元**收购；至 2022 年累计税前利润 **约 20 亿美元**——巴菲特称之为"最有价值的投资课"：品牌定价权 + 零资本开支。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重    | 核心指标                                  | 满分规则                                                       |
| -------- | ------- | ----------------------------------------- | -------------------------------------------------------------- |
| 基本面   | **42%** | ROE、净利增速、毛利率、FCF/净利润、负债率 | ROE≥20%→100分、≥15%→75分、≥10%→40分、<10%→0分                  |
| 估值     | **28%** | PE、PB、安全边际                          | PE≤15→100分、≤25→60分、≤40→25分、>40→0分                       |
| 技术面   | **5%**  | 长期趋势方向                              | 仅参考20日均线方向，上升→60分，震荡→40分，下降→0分             |
| 情绪     | **5%**  | 市场恐慌程度                              | 极度恐慌→100分，正常→50分，亢奋→0分（"别人恐惧我贪婪"）        |
| 安全边际 | **20%** | 负债率、现金流质量、护城河评估            | 负债率<30%且FCF/EPS>0.8→100分，负债率<50%→60分，负债率>70%→0分 |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

- ROE < 10% 或负债率 > 70%（金融业除外）
- FCF 连续 2 年为负
- 公司涉财务造假或管理层失信

### 调用方式

debate 模式中，从 `scripts/data/` 获取最新 ROE/PE/负债率/FCF 后，按上表打分。估值分需结合 PE 历史分位调整——A 股取近 5 年 PE 分位，<20%分位额外+15分，>80%分位额外-20分。

**代码示例：**

```python
from data import get_quote, get_finance
from common import to_float

# 获取数据
quote = get_quote("sh600989")
fin = get_finance("sh600989")[0]

# 基本面维度
roe = to_float(fin.get("ROEJQ", 0))
if roe >= 20:
    base_score = 100
elif roe >= 15:
    base_score = 75
elif roe >= 10:
    base_score = 40
else:
    base_score = 0

# 估值维度
pe = to_float(quote.get("pe", 0))
if pe <= 15:
    pe_score = 100
elif pe <= 25:
    pe_score = 60
elif pe <= 40:
    pe_score = 25
else:
    pe_score = 0

# 安全边际维度
debt_ratio = to_float(fin.get("ZCFZL", 0))
fcf_ratio = to_float(fin.get("MGJYXJJE", 0)) / max(to_float(fin.get("EPSJB", 1)), 0.01)
if debt_ratio < 30 and fcf_ratio > 0.8:
    margin_score = 100
elif debt_ratio < 50:
    margin_score = 60
else:
    margin_score = 0
```
