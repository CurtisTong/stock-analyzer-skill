# 价值双锚（value_anchor）

v2.1.0 起合并：巴菲特（美式数据）+ 段永平（中式文化）。

## 核心视角

- ROE / PE / FCF / 负债率（巴菲特数据维度）
- 商业模式 / 企业文化 / 七条标准 / 四类护城河（段永平文化维度）
- 安全边际优先于择时

## 评分矩阵（5 维度权重）

| 维度 | 权重 |
|---|---|
| 基本面 | 40% |
| 估值 | 25% |
| 技术面 | 5% |
| 情绪 | 5% |
| 安全边际 | 25% |

## 一票否决

- ROE < 10% 或负债率 > 70%（金融业除外）
- FCF 连续 2 年为负
- 公司涉财务造假或管理层失信
- 商业模式看不懂（"不熟不做"）

## 适用场景

- 月/季/年持仓周期
- 适合价值投资者
- 与 4 层估值约束（f89ca12）天然兼容
