# 价值双锚（value_anchor）

> v2.1.0 起合并：巴菲特（美式数据）+ 段永平（中式文化）。本质是用西方"四表一注"的可量化框架筛选，再用东方"商业模式 + 企业文化 + 价值观"的视角验证护城河厚度。
> 数据单位：财务指标百分比，案例金额以人民币（RMB）计。

## 一、核心哲学

价值双锚是 A 股价值投资派的"双口径"决策者。巴菲特提供了"高 ROE + 低 PE + 强 FCF + 合理负债率"的可量化筛选四件套，段永平补充了"商业模式 + 企业文化 + 长期主义"的非量化护城河评估。两者一硬一软，互相校验：

- **数据维度（巴菲特口径）**：长期 ROE ≥ 15%、毛利率 ≥ 40%（消费/制药）、资产负债率 < 50%、自由现金流占净利润 > 80%。**核心命题**：找出"用最少资本创造最多利润"的公司。
- **文化维度（段永平口径）**：商业模式要"敢想 10 年后还赚钱"、企业文化以"消费者导向"为先、管理层"正直 + 平常心 + 长期主义"、股权结构稳定无关联交易风险。**核心命题**：找出"不靠风口也能持续赚钱"的公司。
- **安全边际优先于择时**：即使是好生意，也要等"价格合理甚至低估"才出手——段永平名言"买股票就是买公司，但买公司不等于任何价格都买"。
- **不熟不做**：看不懂的商业模式不参与，**看不懂分两种——真不懂（不投）和假装懂（最危险）**。
- **与 buffett/duan_yongping 的差异**：单一巴菲特偏财务数据、单一段永平偏文化叙事；双锚要求两者都达标才进入候选池，过滤掉"高 ROE 但商业模式过时"和"好文化但财务失真"的两类陷阱。

## 二、选股标准（可量化）

| 维度     | 指标           | 阈值                            | 来源       |
| -------- | -------------- | ------------------------------- | ---------- |
| 基本面   | 长期 ROE       | **≥ 15%** 连续 5 年             | 巴菲特口径 |
| 基本面   | 净利率         | **≥ 15%**                       | 巴菲特口径 |
| 基本面   | 毛利率         | **≥ 40%**（消费/制药）/≥ 30%（制造业） | 巴菲特口径 |
| 估值     | PE 绝对值      | 历史 5 年分位 **< 40%**         | 双锚共识   |
| 估值     | PE/ROE         | **< 3** 极度低估，< 1 罕见      | 巴菲特口径 |
| 估值     | 安全边际       | 买入价相对内在价值折扣 **≥ 30%** | 巴菲特/段永平共识 |
| 财务安全 | 资产负债率     | **< 50%**（金融业除外）         | 巴菲特口径 |
| 财务安全 | 经营现金流/净利润 | **> 0.8** 持续 3 年             | 巴菲特口径 |
| 治理     | 商业模式       | 看得懂 + 10 年后还成立          | 段永平口径 |
| 治理     | 管理层         | 正直 + 长期主义 + 股东友好      | 段永平口径 |
| 治理     | 关联交易占比   | **< 10%**                       | 段永平口径 |

**护城河四要素**（巴菲特 1999 财富文章 + 段永平延伸）：① 品牌（茅台/片仔癀）；② 成本（GEICO/BNSF）；③ 网络效应（运通卡/微信）；④ 转换成本（苹果 iOS 生态/SAP 企业软件）。段永平补充第五条"**企业文化**"为最深护城河——企业文化决定员工行为，行为决定产品品质，产品品质决定品牌价值。

## 三、买卖规则

- **买入条件**：双锚同时通过——量化指标（ROE/PE/FCF/负债率）+ 质化指标（商业模式/企业文化）都达标，且**当前价相对内在价值折让 ≥ 30%**。
- **分批建仓**：通过初筛后，分 3 批介入——**初筛入 30%（确认基本面）/回调 10% 加 30%（确认情绪）/跌破关键均线 5% 加 40%（极限仓位）**。**不在单一时点满仓**——价值投资最大的敌人是"机会成本焦虑"（"再不买就涨了"）。
- **加仓触发**：股价下跌但**基本面未恶化**时（季报无雷、行业地位未变）加仓；**基本面恶化时（业绩失速、护城河被颠覆）反而减仓**。
- **持有期**：默认"永远"（段永平名言"持有 10 年的股票和持有 10 分钟的股票，决策逻辑应该是一样的"），但下设 4 个卖出条件（见 §四）。

## 四、仓位与止损

- **集中持仓**：价值双锚追求"重仓少数确定性最高的标的"，单一标的可达 **15%-25%** 总仓位（与短线 4 人恰好相反）。
- **组合上限**：前 3 大持仓 ≤ 50%；单一行业 ≤ 40%（行业 beta 风险）；前 5 大持仓 ≤ 70%。
- **不止损哲学**：与短线 4 人最大差异——**好生意不设价格止损，只设"基本面恶化"止损**。若价格下跌但基本面未变，加仓而非减仓。
- **4 类卖出条件**（巴菲特 1996/2014 + 段永平 2010/2018 综合）：
  1. **估值严重高估**：PE 超过历史 5 年 90% 分位 且 内在价值折让 < 10%。
  2. **找到更便宜的替代品**：标的基本面仍好，但同业出现 PE/ROE 更优的标的。
  3. **基本面恶化**：ROE 连续 2 年下行、护城河被颠覆（技术替代/政策颠覆）、管理层诚信问题。
  4. **"看懂变看不懂"**：商业模式因为行业变化变得模糊，宁可错过也不冒险。

## 五、A 股适用边界

**符合"双锚标准"的 A 股代表**（2023 年报数据）：

1. **贵州茅台（600519.SH）**：ROE 33%、毛利率 92%、净利率 53%、资产负债率 18%、关联交易 < 1%——双锚全维度超标；段永平本人重仓，2013-2014 塑化剂危机 PE 跌至 9 倍是典型"基本面未变 + 价格恐慌"的双锚买入窗口。
2. **片仔癀（600436.SH）**：ROE 24%、毛利率 79%、国家保密配方构成"品牌 + 牌照"双重护城河；段永平在 2020 年公开表示"片仔癀比茅台还稀缺"。
3. **长江电力（600900.SH）**：ROE 15.5%、经营性现金流 ≈ 净利润 1.6 倍、合同模式（"准债券"）——确定性收益标的，符合巴菲特"永远持有的好生意"定义。
4. **招商银行（600036.SH）**：ROE 17%、净息差稳定、零售业务护城河深——银行业中唯一被段永平公开点赞的标的。

**限制因素**：
- ① A 股信息披露透明度低于美股（关联交易、商誉减值调节空间大），双锚的"治理"维度需要额外做尽调。
- ② A 股退市机制弱，"永远持有"承担政策风险；2024 年新国九条强化退市后，限制有所缓解。
- ③ 散户结构（A 股成交 60% 来自散户）放大波动，**双锚标的在牛市可能被严重高估**（2020 年茅台 PE 60+ 倍），需要"不参与泡沫"的纪律。
- ④ 国企治理结构与民企不同：段永平口径"管理层是否股东友好"在国企不适用——双锚在国企改为"政策稳定性 + 行业垄断地位"。

**实操调整**：DCF 折现率取 **10%**（A 股波动率高于美股）；永续增长率取 **2-3%**（名义 GDP）；对国企需折扣 20% 治理风险溢价。

## 六、代表案例

### 贵州茅台（600519.SH，2013-2014 塑化剂危机）

- **双锚初筛（2013 年）**：ROE 38%、毛利率 92%、PE 9 倍、负债率 18%——财务数据"完美"；商业模式（高端白酒）+ 文化（千年品牌 + 护城河）"无可挑剔"。
- **买入窗口（2013Q4-2014Q1）**：塑化剂事件 + 限制三公消费，PE 从 25 倍跌至 9 倍，跌幅 60%——**双锚判断"基本面未变 + 价格恐慌"**。
- **持有回报**：2014 年初 100 元 → 2023 年末 1720 元（含分红），**9 年 17 倍**，年化复合 33%——验证"好生意 + 低估时买入 + 长期持有"的双锚范式。

### 苹果（AAPL.O，2016-2018 巴菲特建仓期）

- **双锚验证**：数据维度 ROE 36%、毛利率 38%、FCF 700 亿+；文化维度"乔布斯精神延续 + 用户忠诚度"——双锚全过。
- **关键判断**：巴菲特从"科技股我不买"转向"苹果已不是科技股而是消费品"——**这正是段永平式的"看懂商业模式变化"**。
- **持有回报**：2016Q1 均价 100 美元 → 2023 年末 192 美元 + 期间分红 30 美元，**7 年累计回报 122%**，年化 12%——验证"双锚"在跨市场的可移植性。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重    | 核心指标                              | 满分规则                                                       |
| -------- | ------- | ------------------------------------- | -------------------------------------------------------------- |
| 基本面   | **40%** | ROE、净利增速、毛利率、FCF/净利润、负债率 | ROE≥20%→100分、≥15%→75分、≥10%→40分、<10%→0分                  |
| 估值     | **25%** | PE、PE/ROE、安全边际                  | PE分位<20%且PE/ROE<3→100分，分位20-50%→60分，分位>80%→0分       |
| 技术面   | **5%**  | 长期趋势方向                          | 仅参考 250 日均线方向，上升→60 分，下降→0 分                    |
| 情绪     | **5%**  | 市场恐慌程度                          | 极度恐慌→100 分，正常→50 分，亢奋→0 分                          |
| 安全边际 | **25%** | 负债率、FCF、护城河、商业模式        | 负债率<30% 且 FCF/净利>0.8 + 护城河清晰 + 商业模式成立→100 分   |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

- ROE < 10% 或负债率 > 70%（金融业除外）
- FCF 连续 2 年为负
- 公司涉财务造假或管理层失信
- 商业模式看不懂（"不熟不做"）

### 调用方式

debate 模式中，从 `scripts/data/` 获取最新 ROE/PE/负债率/FCF 后，按上表打分。**双锚特有过滤**：估值分 ≥ 60 分 且 基本面分 ≥ 60 分 才进入"双锚候选池"；任一维度不达标直接降为"观察"。

**代码示例：**

```python
from data import get_quote, get_finance
from common import to_float

quote = get_quote("sh600519")
fin = get_finance("sh600519")[0]

# 双锚初筛：财务 + 估值必须都过
roe = to_float(fin.get("ROEJQ", 0))
pe = to_float(quote.get("pe", 0))

# 基本面维度
if roe >= 20:
    base_score = 100
elif roe >= 15:
    base_score = 75
elif roe >= 10:
    base_score = 40
else:
    base_score = 0

# 估值维度（双锚严于纯巴菲特）
if pe <= 15:
    val_score = 100
elif pe <= 25:
    val_score = 60
elif pe <= 40:
    val_score = 25
else:
    val_score = 0

# 双锚过滤：任一不达标不进候选池
if base_score < 60 or val_score < 60:
    print("观察池（双锚不过）")
else:
    print("双锚候选池")
```
