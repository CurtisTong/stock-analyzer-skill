## 长线内部差异

- **巴菲特 vs 段永平**：巴菲特对价格更敏感（"用合理价格买伟大公司"），段永平更注重商业模式与文化。
- **巴菲特 vs 林奇**：巴菲特偏好稳定可预测的现金流，林奇偏好高增速、可被散户生活洞察到的成长股。
- **林奇 vs 索罗斯**：林奇是个股 Alpha 选手，索罗斯是宏观 Beta + 反身性，方法论层面差异最大。

## 短线内部差异

- **徐翔 vs 赵老哥**：徐翔是当日打板次日走，赵老哥是二板定龙头后波段持有。
- **赵老哥 vs 作手新一**：赵老哥是强者追涨，作手新一是分歧低吸。
- **炒股养家 vs 其余三者**：养家最重情绪周期判断，选股反而是其次；其他三人更重具体形态与量价。

## 评分与决策体系

每位专家末尾均已追加 **§九、评分矩阵**，包含：

- 5 维度评分卡（基本面、估值、技术面、情绪/题材、风险）
- 自定义权重（如巴菲特重基本面 42%+安全边际 20%，养家重情绪 65%）
- 一票否决条件
- 数据调用方式

**决策引擎**：[decide.md](decide.md) 整合 8 人评分：

- 投票机制：分组计票 + 加权综合分
- 市场环境动态调权：牛市长线40%/短线60%，熊市长线65%/短线35%
- 冲突解决：巴菲特否决权、养家情绪周期降权
- 信心指数：基于专家一致性和综合分

## 快速使用指南

**debate 模式支持三种子模式**：

```text
/stock <名称> debate          # 全模式（默认，8人跨组加权）
/stock <名称> debate 长线     # 仅长线4人（巴菲特/林奇/索罗斯/段永平）
/stock <名称> debate 短线     # 仅短线4人（徐翔/赵老哥/养家/作手新一）
```

**标准流程**：

1. 获取目标股票数据（`scripts/data/`）
2. 获取大盘环境数据（`scripts/quote.py sh000001,sh510300` 或 `get_quote("sh000001")`）
3. 判断市场状态 → 确定长线/短线权重
4. 每位专家按 §九 评分矩阵独立打分
5. 按 decide.md 进行投票汇总和冲突解决
6. 输出：评分表 + 方向 + 风险 + 仓位

**代码示例**：

```python
from data import get_quote, get_quotes, get_kline, get_finance

# 获取单只股票数据
quote = get_quote("sh600989")
finance = get_finance("sh600989")
kline = get_kline("sh600989", scale=240, datalen=30)

# 获取多只股票数据
quotes = get_quotes(["sh600989", "sz000807"])

# 获取大盘数据
index_quote = get_quote("sh000001")  # 上证指数
etf_quote = get_quote("sh510300")    # 沪深300 ETF
```

## 合并型专家评分模式

v2.1.0 引入的合并型专家（value_anchor/topic_leader/emotion_tech）使用 `weighted_merge()` 模式：

- 内部调用多个 legacy 专家的 score() 函数
- 按预定义权重加权平均
- 简洁（每个约 20 行），逻辑透明

新增合并型专家时遵循此模式。

## 使用建议

- **/stock 完整分析**：长线 4 人给出价值锚点，短线 4 人给出时机建议。
- **多空辩论**：每位专家从自身框架独立打分，由 [decide.md](decide.md) 汇总。
- **A 股适配**：所有短线策略均明确标注 T+1 / 涨跌停 / 量化竞争等约束。
- **市场环境优先**：选策略 → 选专家权重 → 选股，不要跳过市场判断。
