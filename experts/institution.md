# 机构派（institution）

> v2.1.0 起新增：填补机构长期主义视角盲区。本质是用高瓴张磊、红杉沈南鹏、淡马锡等长线机构的"深度尽调 + 长期持有 + 集中持仓"框架，过滤短期波动噪音。
> 数据来源：上市公司年报与季报、机构调研记录、基金持仓公告（沪深交易所定期披露）、行业研究报告。金额单位人民币（RMB）。

## 一、核心哲学

机构派是通用 8 人专家体系（5 长线 + 3 短线）的"长期主义"补丁。通用专家（巴菲特/林奇等）虽也强调长期持有，但**机构派的"长期"是 5-10 年维度的产业投资**，而非巴菲特 3-5 年的财务投资。机构派核心逻辑：

- **深度尽调（Deep Diligence）**：高瓴张磊名言"做时间的朋友"，但前提是**做了 3-6 个月的尽调**——管理层访谈 50+ 次、行业专家访谈 30+ 次、客户访谈 20+ 次、供应商访谈 15+ 次。
- **长期持有（Patient Capital）**：红杉沈南鹏"投资就是投人"，5-10 年陪伴创始人成长，淡马锡平均持有期 8-10 年（vs 公募基金 1-2 年）。
- **集中持仓（Concentrated Portfolio）**：高瓴前 10 大持仓占组合 60%+（vs 公募基金通常 30%），**集中持仓的前提是尽调深度**。
- **公司治理 + 管理层诚信**：机构派最看重的"软指标"——管理层是否股东友好、是否有诚信问题、是否有长期主义思维。
- **行业空间天花板**：机构派投的是"大赛道"——只投千亿级市场中的龙头（如新能源、半导体、医疗），不投细分小赛道。
- **与通用专家的差异**：
  - **巴菲特**：3-5 年财务投资，看 ROE/PE/FCF。
  - **机构派**：5-10 年产业投资，看公司治理 + 行业空间 + 管理层。
  - **林奇**：散户生活洞察的成长股投资。
  - **机构派**：机构深度尽调的长期成长股投资。
  - **核心区分**：机构派愿意为"长期确定性"支付**估值溢价**（PE 30-50 倍可接受），但要求**长期复合增长 20%+**。

## 二、选股标准（可量化）

| 维度     | 指标                | 阈值                              | 逻辑                       |
| -------- | ------------------- | --------------------------------- | -------------------------- |
| 行业空间 | TAM（总可寻址市场） | **≥ 1000 亿元**（5 年后）         | 大赛道才有龙头机会         |
| 行业空间 | 行业增速            | **≥ 15%**（未来 5 年 CAGR）       | 增量市场才能产生 alpha     |
| 竞争格局 | 市占率              | 行业前 3（最好第 1）              | 龙头护城河                 |
| 竞争格局 | CR3 集中度          | **≥ 30%**                         | 格局已定，非百团大战        |
| 公司治理 | 管理层背景          | 行业经验 ≥ 10 年 + 创始团队稳定   | 长期主义                   |
| 公司治理 | 股东结构            | 创始人/管理层持股 ≥ 10%          | 与股东利益一致             |
| 公司治理 | 关联交易占比        | **< 5%**                          | 治理透明                   |
| 公司治理 | 商誉/净资产         | **< 30%**                         | 收购历史干净               |
| 财务     | 营收增速            | **≥ 20%**（连续 3 年）            | 长期增长                   |
| 财务     | ROE                | **≥ 15%**（连续 3 年）            | 资本回报                   |
| 财务     | FCF/净利润          | **≥ 0.8**                         | 利润含金量                 |
| 估值     | PE                | **≤ 行业历史 5 年 80% 分位**      | 不强求低估，但要合理       |
| 估值     | PEG               | **≤ 1.5**                         | 增速可消化估值             |
| 估值     | DCF 折现           | 当前价 ≤ 5 年后合理价值 × 0.7     | 机构派愿意持有 5 年        |

**机构派"三好"选股标准**（高瓴张磊《价值》总结）：
1. **好生意**：行业空间大 + 商业模式可持续 + 护城河深。
2. **好公司**：管理层优秀 + 公司治理透明 + 长期主义。
3. **好价格**：DCF 折现合理（机构派愿意持有 5-10 年，所以对短期估值容忍度高）。

## 三、买卖规则

- **买入条件**：行业空间 + 竞争格局 + 公司治理 + 财务四维度都达标，且当前价相对 DCF 5 年合理价值折让 ≥ 20%（机构派容忍度高于巴菲特的 30% 折让，因为持有期更长）。
- **建仓节奏**：机构派追求"重仓少数确定性最高的标的"，**单一标的可达 10%-20% 总仓位**——但建仓期长（3-6 个月分批），避免追高。
- **加仓触发**：股价下跌但**基本面与行业趋势未变**时加仓；**基本面恶化时（业绩失速、护城河被颠覆、管理层诚信问题）反而减仓**。
- **持有期**：默认 **5-10 年**（远长于巴菲特 3-5 年的"永远"），但下设 4 个卖出条件（见 §四）。
- **不参与短期博弈**：机构派不参与主题炒作、情绪博弈、短线题材——**对市场短期波动"钝感"**。

## 四、仓位与止损

- **集中持仓**：单一标的 **10%-20%**；前 3 大持仓 ≤ 50%；前 5 大持仓 ≤ 70%——**集中度高于巴菲特的"永远持有少数"框架**。
- **组合上限**：单一行业 ≤ 40%；前 3 大行业 ≤ 70%。
- **不止损哲学**：与价值双锚一致——**好生意不设价格止损，只设"基本面恶化"止损**。若价格下跌但基本面与行业趋势未变，加仓而非减仓。
- **4 类卖出条件**：
  1. **公司治理恶化**：管理层诚信问题、关联交易激增、股东结构不稳。
  2. **行业空间见顶**：TAM 增长停滞、竞争格局恶化（龙头被颠覆）。
  3. **基本面失速**：营收增速连续 2 季 < 10% 或 ROE 连续 2 季下行。
  4. **估值极度高估**：PE 超过历史 5 年 95% 分位（机构派对估值容忍度高，但极端高估仍要减仓）。

## 五、A 股适用边界

**符合"机构派"框架的 A 股代表**（2023-2024 数据）：

1. **宁德时代（300750.SZ）**：动力电池全球市占率 35% + 储能业务高速增长 + 管理层曾毓群深耕电池 30 年——**高瓴 2020 年定增 100 亿入股，3 年浮盈 50%**。
2. **迈瑞医疗（300760.SZ）**：医疗器械国产替代 + 海外市场扩张 + 管理层李西廷深耕医疗器械 30 年——**高瓴 / 红杉多次调研，机构持仓占比 30%+**。
3. **药明康德（603259.SH）**：CXO 全球市占率前三 + 管理层李革深耕 CRO 20+ 年——**淡马锡持仓多年，机构派典型长期持有**。
4. **海康威视（002415.SZ）**：安防全球市占率第一 + 管理层胡扬忠深耕安防 20+ 年——**高瓴 / 淡马锡长期持仓，机构派典型**。
5. **贵州茅台（600519.SH）**：高端白酒绝对龙头 + 管理层稳定 + 现金流好——**机构派 + 巴菲特 + 段永平 共识标的**。

**限制因素**：
- ① A 股机构资金占比 30%-40%（vs 美股 60%+），**散户结构放大短期波动**，机构派的"长期持有"在 A 股需要更强纪律。
- ② A 股信息披露透明度低于美股，**深度尽调的成本更高**——机构派在 A 股的尽调周期通常比美股长 50%。
- ③ A 股"壳价值"与"题材炒作"文化深厚，**机构派的"钝感"在 A 股是优势也是劣势**（优势：不被噪音干扰；劣势：可能错过主题机会）。
- ④ 政策风险：行业政策变化（如双碳、医保集采）对长期持有有颠覆性影响，**机构派在政策敏感行业的尽调需加倍**。
- ⑤ 实操调整：DCF 折现率取 **10%**（A 股波动率高于美股）；永续增长率取 **2-3%**（名义 GDP）；对国企需折扣 20% 治理风险溢价。

## 六、代表案例

### 宁德时代（300750.SZ，2020-2024 高瓴重仓）

- **深度尽调（高瓴 2018-2020）**：
  - 管理层访谈 50+ 次：曾毓群深耕电池 30 年，工程师文化强。
  - 行业专家访谈 30+ 次：动力电池行业 5 年 CAGR 30%+，确定性高。
  - 客户访谈 20+ 次：特斯拉 / 宝马 / 大众等头部车企深度合作。
  - 供应商访谈 15+ 次：上游锂矿 / 隔膜 / 电解液供应链稳定。
- **核心判断**：
  - 好生意：动力电池 TAM 5 万亿 + 行业增速 30% + 龙头护城河。
  - 好公司：管理层长期主义 + 公司治理透明 + 国产替代受益。
  - 好价格：2020 年定增价 161 元，DCF 5 年合理价值 250 元，折让 35%。
- **持仓回报**：
  - 高瓴 2020 年 7 月 100 亿入股（161 元/股）。
  - 2021 年末股价最高 382 元（+137%）。
  - 2024 年中股价 200 元（+24%，考虑分红再投入）。
  - **机构派验证**：长期持有 4 年，回报远高于短期博弈。

### 迈瑞医疗（300760.SZ，2018-2024 高瓴 + 红杉联合持仓）

- **深度尽调（2018 年）**：
  - 管理层访谈：李西廷深耕医疗器械 30 年，技术导向。
  - 行业专家访谈：医疗器械国产替代率从 30% 提升至 70%。
  - 客户访谈：国内三甲医院覆盖 + 海外市场拓展。
- **核心判断**：
  - 好生意：医疗器械 TAM 1 万亿 + 行业增速 15% + 龙头护城河。
  - 好公司：管理层长期主义 + 研发投入 10%+ + 海外市场扩张。
  - 好价格：2018 年 PE 50 倍，PEG 1.0（增速可消化估值）。
- **持仓回报**：
  - 高瓴 / 红杉 2018 年起持续加仓。
  - 2024 年中股价相对 2018 年 +200%（含分红再投入）。
  - **机构派验证**：长期持有 6 年，复合回报 20%+。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重    | 核心指标                          | 满分规则                                                       |
| -------- | ------- | --------------------------------- | -------------------------------------------------------------- |
| 基本面   | **50%** | 行业空间、竞争格局、营收增速、ROE | 行业空间 ≥ 1000 亿 + 市占率前 3 + 营收增速 ≥ 20% + ROE ≥ 15% →100 分 |
| 估值     | **20%** | PE/PEG、DCF 折让                  | PEG ≤ 1.0 + DCF 折让 ≥ 30% →100 分，PEG 1.0-1.5 →60 分，PEG > 2 →0 分 |
| 技术面   | **5%**  | 长期趋势方向                      | 仅参考 250 日均线方向，上升→60 分，下降→0 分                    |
| 情绪     | **5%**  | 机构持仓变化                      | 机构持仓环比+5% →100 分，机构持仓-5% →0 分                     |
| 安全边际 | **20%** | 公司治理、关联交易、商誉          | 治理透明 + 关联交易 < 5% + 商誉/净资产 < 30% →100 分             |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

- 公司治理结构有问题（管理层失信、关联交易激增）
- 行业空间天花板可见（5 年 TAM 增长 < 5%）
- 管理层短视（季度业绩导向、不做长期投入）
- 商誉/净资产 > 50%（并购历史不干净）
- 创始人/管理层持股 < 5%（与股东利益不一致）

### 调用方式

debate 模式中，从 `scripts/data/` 获取营收增速 / ROE / 关联交易 / 商誉占比后，按上表打分。**机构派特有过滤**：基本面分 ≥ 60 + 治理分 ≥ 60 才进候选池；任一不达标直接降为"观察"。

**代码示例：**

```python
from data import get_quote, get_finance
from common import to_float

quote = get_quote("sz300750")  # 宁德时代
fin = get_finance("sz300750")[0]

# 机构派：核心看基本面 + 治理
rev_growth = to_float(fin.get("TOTALOPERATEREVETZ", 0))
roe = to_float(fin.get("ROEJQ", 0))

# 基本面维度
if rev_growth >= 20 and roe >= 15:
    base_score = 100
elif rev_growth >= 10 and roe >= 10:
    base_score = 60
else:
    base_score = 0

# 估值维度（机构派容忍度高于巴菲特）
pe = to_float(quote.get("pe", 0))
peg = pe / rev_growth if rev_growth > 0 else 999
if peg <= 1.0:
    val_score = 100
elif peg <= 1.5:
    val_score = 60
else:
    val_score = 0

# 机构派特有过滤
if base_score < 60 or val_score < 60:
    print("观察池（机构派不过）")
else:
    print("机构派候选池")
```
