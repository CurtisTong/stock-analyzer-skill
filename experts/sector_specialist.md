# 行业专家（sector_specialist）

> v2.1.0 起新增：填补行业特异性视角盲区。本质是用行业景气度 + 竞争格局 + 估值分位的差异化阈值，识别"同一财务指标在不同行业的含义截然不同"导致的误判。
> 数据来源：行业指数（东方财富/同花顺行业板块）、行业协会数据、Wind 行业研究报告。金额单位人民币（RMB）。

## 一、核心哲学

行业专家是通用 8 人专家体系的"行业特异性"补丁。通用专家（巴菲特/林奇/段永平等）用一套统一的财务阈值筛选标的，但实际投资中**同一指标在不同行业的含义差异巨大**：

- **PE**：消费股 30 倍合理，半导体 60 倍正常，银行 5-10 倍才是低估。
- **ROE**：地产 8% 优秀（重资产行业），消费 15% 才入门，券商 ROE 波动巨大需看均值。
- **增速**：创新药 50% 增速是起步（临床后期放量），成熟消费股 20% 已是高速。
- **负债率**：银行 90% 正常（吸收存款），消费股 > 50% 就有风险。

**行业专家的核心命题**：**用"行业 + 标的"双维度筛选，避免"用消费股标准筛地产股"的张冠李戴**。

- **行业景气度判断**：医药/科技/消费/周期/金融五大类的子行业景气度差异化（详见 §二）。
- **竞争格局识别**：龙头 / 二线 / 细分龙头 / 尾部分化——同一财务指标在不同竞争位置的意义不同。
- **估值分位（行业 PE/PB 历史分位）**：相比绝对 PE，**行业历史分位更稳定**（不同时点跨行业比较不可靠）。
- **技术替代风险识别**：行业专家最重视"颠覆性技术"风险——Nokia / 柯达 / 柯达 / 雅虎都是技术替代受害者。
- **与通用专家的差异**：通用专家用一套阈值（如 PE<25），行业专家**根据子行业调整阈值**（半导体 PE<60、消费 PE<25、银行 PE<8）。同一只股票，通用专家可能给"中性"，行业专家给"强烈看多"（如创新药 PE 50 倍在通用专家眼中偏高，但行业专家知道创新药估值贵但代表未来增长）。

## 二、选股标准（可量化，按子行业差异化）

### 1. 消费类（白酒/食品/家电/医药消费）

| 指标         | 阈值                              | 逻辑                       |
| ------------ | --------------------------------- | -------------------------- |
| ROE          | **≥ 15%**                         | 品牌护城河可承受高 ROE     |
| 毛利率       | **≥ 40%**（白酒/食品）/≥ 60%（高端白酒） | 品牌定价权 |
| 净利率       | **≥ 15%**                         | 品牌+渠道护城河            |
| PE           | **< 30 倍**（白酒可至 35 倍）     | 确定性溢价                 |
| 增速         | 营收 ≥ 10%、净利 ≥ 15%            | 稳定增长                   |
| 负债率       | **< 50%**                         | 轻资产模式                 |

**代表标的**：贵州茅台（600519）、五粮液（000858）、海天味业（603288）、伊利股份（600887）、美的集团（000333）。

### 2. 科技类（半导体/软件/互联网/AI）

| 指标         | 阈值                              | 逻辑                       |
| ------------ | --------------------------------- | -------------------------- |
| ROE          | **≥ 10%**（早期可放宽）           | 投入期 ROE 不强求          |
| 营收增速     | **≥ 30%**（半导体）/≥ 50%（AI 软件） | 行业天花板未到            |
| 研发占比     | **≥ 10%**                         | 持续创新                   |
| PE           | **< 60 倍**（早期可至 100 倍）    | 高增长高估值可接受         |
| PS（市销率） | **< 20 倍**（SaaS 类）            | 早期无盈利用 PS 替代       |
| 自由现金流   | 转正或接近转正                     | 商业模式可持续性           |

**代表标的**：海光信息（688041）、中芯国际（688981）、金山办公（688111）、科大讯飞（002230）、迈瑞医疗（300760）。

### 3. 医药类（创新药/医疗器械/CXO/中药）

| 指标         | 阈值                              | 逻辑                       |
| ------------ | --------------------------------- | -------------------------- |
| 创新药       | 临床 III 期数据 + BD 能力         | 未来现金流折现            |
| CXO          | 在手订单 ≥ 2 年营收               | 行业景气度直接指标         |
| 医疗器械    | 国产替代率 + 集采免疫度           | 政策风险差异化             |
| 中药         | 品牌 + 保密配方                   | 消费品逻辑                 |
| 增速         | **≥ 30%**（创新药）/≥ 15%（中药） | 子行业差异巨大             |

**代表标的**：恒瑞医药（600276）、药明康德（603259）、迈瑞医疗（300760）、片仔癀（600436）、百济神州（688235）。

### 4. 周期类（有色/化工/钢铁/煤炭/航运）

| 指标         | 阈值                              | 逻辑                       |
| ------------ | --------------------------------- | -------------------------- |
| ROE          | **≥ 行业历史 70% 分位**           | 周期顶部特征               |
| 商品价格     | 处于历史 50-80% 分位              | 顺周期买入窗口             |
| 库存周期     | 主动去库 → 被动去库转换点         | 周期拐点                   |
| PE           | **< 行业历史 50% 分位**           | 周期底部估值               |
| 负债率       | **< 60%**                         | 周期下行时去杠杆保护       |
| 股息率       | **≥ 4%**                          | 周期股安全垫               |

**代表标的**：紫金矿业（601899）、洛阳钼业（603993）、中国神华（601088）、海螺水泥（600585）、中远海控（601919）。

### 5. 金融类（银行/保险/券商）

| 指标         | 阈值                              | 逻辑                       |
| ------------ | --------------------------------- | -------------------------- |
| 银行 PB      | **< 0.7**（国有大行）/ < 1.0（股份行） | 银行破净是常态             |
| 银行 ROE     | **≥ 10%**                         | 资本回报稳定               |
| 银行不良率   | **< 1.5%**                        | 资产质量                   |
| 银行拨备覆盖率 | **≥ 150%**                       | 风险缓冲                   |
| 券商 ROE     | **≥ 行业历史 70% 分位**           | 周期性强                   |
| 保险 PEV     | **< 0.7**                         | 内含价值折让               |

**代表标的**：招商银行（600036）、宁波银行（002142）、兴业银行（601166）、中信证券（600030）、中国平安（601318）。

## 三、买卖规则

- **行业景气拐点买入**：从景气度低点向高点转换时（如 2024 年初的医药、半导体），即使个股基本面未明显改善，也可分批建仓。
- **行业景气顶点减仓**：从景气度高点向低点转换时（如 2021 年的部分周期股），即使个股 PE 仍低，也应分批减仓。
- **子行业切换**：同一行业大类内部切换（如消费 → 必选消费 → 高端消费）——消费整体降级时，必选消费（食品/日化）反而抗跌。
- **政策催化跟踪**：行业专家对政策敏感度最高——医保集采、双碳目标、半导体大基金等政策变化时，第一时间调整子行业排序。
- **避免"夕阳行业陷阱"**：成熟期 + 增速持续下行 + 技术替代风险（如传统燃油车、传统纸媒）——即使估值便宜也不参与。

## 四、仓位与止损

- **子行业集中度**：单一子行业 ≤ 30% 总仓位（避免子行业系统性风险）。
- **行业大类分散**：同时配置 2-3 个不同行业大类（如消费+科技+周期），降低行业 beta 风险。
- **止损**：
  - **基本面止损**：行业景气度从上行转下行、个股 ROE 连续 2 季下滑、竞争格局恶化（龙头被颠覆）→ 减仓。
  - **估值止损**：行业 PE/PB 历史分位 > 90% → 减仓（高估）。
  - **政策止损**：监管政策变化对子行业有重大影响（如集采扩围）→ 重新评估。
- **周期股止损**：与成长股不同——**周期股 PE 高反而是买入信号**（行业顶部前），PE 低反而是卖出信号（行业底部前）；需结合商品价格 + 库存周期综合判断。
- **不加杠杆**：行业系统性风险大，加杠杆会放大 beta。

## 五、A 股适用边界

**符合"行业专家"框架的 A 股代表**（2023-2024 数据）：

1. **紫金矿业（601899.SH）**：铜金双极 + 全球并购扩张，**周期类典型**——2024 年铜价处于历史 70% 分位 + 公司全球储量增长 30%，符合"周期景气上行 + 龙头扩张"双驱动。
2. **海光信息（688041.SH）**：国产替代 + AI 算力需求，**科技类典型**——2024 年 DCU 国产替代率从 10% 提升至 30% + AI 算力需求增长 50%，符合"科技景气上行 + 国产替代"双驱动。
3. **恒瑞医药（600276.SH）**：创新药 BD + 集采影响分化，**医药类典型**——2024 年创新药收入占比从 30% 提升至 50% + 集采影响减弱，符合"创新药转型 + 政策风险消化"双驱动。
4. **招商银行（600036.SH）**：零售护城河 + 资产质量优，**金融类典型**——2024 年零售客户 AUM 增长 15% + 不良率 0.95% 稳定，符合"零售护城河 + 资产质量"双驱动。

**限制因素**：
- ① 行业轮动极快：单一行业景气度周期 2-3 年（A 股）vs 美股 5-7 年，**对跟踪频率要求高**。
- ② 政策风险：中国行业受政策影响远大于美股，**集采、双碳、芯片大基金等政策变化随时颠覆行业逻辑**。
- ③ 子行业分化：同行业大类内部子行业走势分化极大（如医药整体下跌时，中药反而上涨），**通用专家无法替代行业专家的精度**。
- ④ 估值体系复杂：不同子行业的估值锚不同（PE/PB/PS/PEG/PEV），**没有一个通用估值公式**。
- ⑤ 适用窗口：**全市场周期均适用**（无论牛熊）——行业专家不依赖大盘判断，依赖子行业景气度判断。

## 六、代表案例

### 紫金矿业（601899.SH，2024 周期反转）

- **行业判断（2023Q4）**：铜价处于历史 50% 分位 + 全球库存去化 + 美元加息周期见顶——**周期景气拐点**。
- **公司判断**：全球铜矿储量第三 + 成本控制能力强 + 2024 年并购放量——**周期龙头**。
- **买入逻辑**：周期景气拐点 + 龙头公司 + 估值合理（PE 15 倍）——行业专家强烈看多。
- **回报验证（2024 H1）**：紫金矿业股价 +35%，跑赢沪深 300 指数 25 个百分点——验证"行业专家 + 周期反转"框架。

### 海光信息（688041.SH，2024 AI 算力）

- **行业判断（2023Q4）**：美国出口管制 + 国产替代窗口 + AI 算力需求爆发——**科技 + 国产替代双驱动**。
- **公司判断**：海光 DCU 性能对标英伟达 A100 的 70% + 国产替代率从 10% 提升至 30%——**科技龙头**。
- **买入逻辑**：科技景气 + 国产替代 + 龙头公司 + 估值合理（PS 30 倍）——行业专家强烈看多。
- **回报验证（2024 H1）**：海光信息股价 +60%，跑赢科创 50 指数 40 个百分点——验证"行业专家 + 国产替代"框架。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重    | 核心指标                          | 满分规则                                                       |
| -------- | ------- | --------------------------------- | -------------------------------------------------------------- |
| 基本面   | **30%** | 子行业差异化财务指标              | 按子行业差异化阈值（消费 15%/科技 30%/医药 30%/周期 70% 分位/银行 10%），达标→100 分 |
| 估值     | **30%** | 行业历史 PE/PB 分位               | 行业历史分位<20% →100 分，20-50% →60 分，>80% →0 分            |
| 技术面   | **10%** | 行业指数趋势、个股相对强度        | 行业指数>MA20 + 个股相对强度>1 →100 分，行业指数<MA20 →40 分   |
| 情绪     | **10%** | 行业 ETF 资金流向、机构持仓        | 行业 ETF 连续 5 日净流入 + 机构持仓环比+5% →100 分              |
| 风险     | **20%** | 政策风险、技术替代风险、子行业景气 | 政策友好+无技术替代+子行业景气上行 →100 分，监管打压 →0 分     |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

- 行业周期顶点（PE/PB 双高 + 增速下行）
- 政策风险（行业被监管打压，如教育培训、互联网平台经济）
- 技术替代风险（核心技术被颠覆，如传统燃油车 vs 新能源、传统手机 vs 智能手机）
- 子行业景气度明确下行（连续 2 季行业指数 < MA20）

### 调用方式

debate 模式中，先判断标的对应的子行业大类（消费/科技/医药/周期/金融/其他），再按子行业差异化阈值打分。**行业专家特有过滤**：估值分（行业历史分位）≥ 60 + 风险分 ≥ 60 才进候选池。

**代码示例：**

```python
from data import get_quote, get_finance
from common import to_float

quote = get_quote("sh600519")  # 贵州茅台 = 消费类
fin = get_finance("sh600519")[0]

# 行业专家：识别子行业
sector = "消费类"  # 需根据标的行业分类确定

# 消费类差异化阈值
if sector == "消费类":
    roe_threshold = 15
    gross_margin_threshold = 40
    pe_threshold = 30
elif sector == "科技类":
    roe_threshold = 10
    growth_threshold = 30
    pe_threshold = 60
# ... 其他子行业

# 基本面维度（按子行业差异化阈值）
roe = to_float(fin.get("ROEJQ", 0))
gross_margin = to_float(fin.get("XSMLL", 0))

if roe >= roe_threshold and gross_margin >= gross_margin_threshold:
    base_score = 100
elif roe >= roe_threshold * 0.8:
    base_score = 60
else:
    base_score = 0

# 估值维度（行业历史分位，简化版用绝对 PE）
pe = to_float(quote.get("pe", 0))
if pe <= pe_threshold * 0.7:
    val_score = 100
elif pe <= pe_threshold:
    val_score = 60
else:
    val_score = 0

# 行业专家特有过滤
if val_score < 60:
    print("观察池（行业估值分位偏高）")
else:
    print(f"{sector}候选池")
```
