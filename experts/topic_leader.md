# 题材龙头（topic_leader）

> v2.1.0 起合并：徐翔（涨停板战法）+ 赵老哥（趋势龙头）。本质是用徐翔的"涨停基因 + 封单强度"识别短线龙头，再以赵老哥的"均线系统 + 题材容量 + 龙虎榜"判断二板定龙头后的波段持有。
> 数据来源：龙虎榜（东方财富/同花顺）、淘股吧涨停板复盘、雪球龙虎榜整理。金额单位人民币（RMB）。

## 一、核心哲学

题材龙头是 A 股短线博弈的"龙头战法合并者"。徐翔代表了"当日封板 + 次日高开接力"的极短打法，赵老哥代表了"二板定龙头 + 波段持有"的中短打法。题材龙头要求两者兼具——既要徐翔的"快进快出执行力"，也要赵老哥的"题材生命周期判断"。

- **题材稀缺性 > 个股本身**：游资圈共识"龙头之所以是龙头，是因为题材稀缺"。同一题材里，资金只会选 1-2 个龙头，**进不去龙头的资金只能去二线跟风**。
- **涨停基因 + 板块联动（徐翔维度）**：个股必须有涨停基因（近 30 日 ≥ 1 次涨停），且涨停时板块效应明显（同期同板块 ≥ 3 只涨停）——孤板无联动的不参与。
- **均线系统 + 题材容量 + 龙虎榜（赵老哥维度）**：5/10/20 日均线多头排列、题材容量大（板块日成交 > 100 亿）、龙虎榜买一席位有"游资共振"（非机构主导）。
- **持仓周期 1-15 日**：与短线 4 人其他三人不同，题材龙头**默认持股 3-5 日**（二板后持有到题材高潮），不参与超短 T+0。
- **与 xu_xiang/zhao_laoge 的差异**：单一徐翔偏"首板后追涨次日走"、单一赵老哥偏"二板后趋势持有"；题材龙头要求"首板敢追、二板敢加、高潮敢走"三位一体，过滤掉"追涨被套"和"持有过气"的两类陷阱。

## 二、选股标准（可量化）

| 维度     | 指标              | 阈值                              | 来源       |
| -------- | ----------------- | --------------------------------- | ---------- |
| 涨停基因 | 近 30 日涨停数    | **≥ 1 次**（连板 ≥ 4 板为强龙头） | 徐翔口径   |
| 涨停基因 | 涨停次日溢价      | **平均 ≥ 3%**（资金接力意愿）     | 徐翔口径   |
| 题材联动 | 涨停当日板块涨停数 | **≥ 3 只**（板块效应）           | 徐翔口径   |
| 题材联动 | 板块当日成交额    | **≥ 100 亿**（题材容量大）        | 赵老哥口径 |
| 趋势系统 | MA5/MA10/MA20     | **多头排列**（MA5>MA10>MA20）     | 赵老哥口径 |
| 趋势系统 | 涨停后 5 日累计涨幅 | **≥ 15%**（趋势确认）             | 赵老哥口径 |
| 龙虎榜   | 买一席位游资集中度 | **净买入 > 3000 万**              | 赵老哥口径 |
| 龙虎榜   | 机构净买入         | **> 0**（机构共振加分）           | 赵老哥口径 |
| 流动性   | 流通市值          | **30-200 亿**（避免小票炸板）     | 共识       |
| 流动性   | 涨停封单/流通市值  | **≥ 0.5%**（封单稳定）            | 徐翔口径   |

**龙头识别四步法**：
1. **首板日**：板块内 ≥ 3 只涨停 + 个股封单稳定 + 龙虎榜有知名游资。
2. **次日高开 ≥ 3%**：资金接力意愿确认（不接力即淘汰）。
3. **二板定龙头**：板块内唯一二板且放量（量能 > 首板 80%）。
4. **三板加速**：缩量加速或放量分歧均可，但必须"不破 5 日线"。

## 三、买卖规则

- **首板介入（徐翔）**：板块首板出现后，**只买板块龙头**（不跟风二线）；打板价买入 30% 仓；**封单不稳定的（炸板率 > 30%）不参与**。
- **二板加仓（赵老哥）**：二板定龙头后加 30% 仓；放量二板（量能 > 首板 80%）优先，缩量二板次之。
- **三板持有 / 高潮减仓**：三板后持有 2-3 日观察题材持续性，**高潮日（板块涨停 > 10 只或个股封单 > 流通市值 1%）减仓 50%**。
- **止损铁律**：**首板介入后 30 分钟内不封板 → 立即出局**（避免"假封");**二板后跌破 5 日线 → 减仓 50%**;**三板后跌破 10 日线 → 清仓**。
- **时间止损**：持股 5 日内未达预期（涨幅 < 5%），无论盈亏减仓 50%；题材龙头节奏快，**抱死是最大风险**。

## 四、仓位与止损

- **单票 10%-20%**：龙头波动大，仓位需控制。
- **分 3 批建仓**：30%（首板打板）+ 30%（二板加仓）+ 40%（题材确认后满仓）。
- **止损**：
  - 首板后 30 分钟未封板 → -3% 止损
  - 二板后跌破 5 日线 → -5% 止损
  - 三板后跌破 10 日线 → -8% 止损
  - 题材退潮（板块涨停 < 3 只持续 2 日）→ 清仓
- **不加仓**：被套后不补仓，**龙头战法的核心是"错了就认"**。
- **总仓位**：题材活跃期（涨停 > 80 只/日）可至 70%；退潮期（涨停 < 30 只/日）降至 30% 以下。

## 五、A 股适用边界

**符合"题材龙头"的 A 股代表**（2022-2024 年案例）：

1. **浙江建投（002761.SZ，2022 基建龙头）**：2022/2/7 - 2/21 连拉 7 板，2/22 首阴后二波 109% 涨幅——徐翔"首板基因"+ 赵老哥"二板定龙头"典型。
2. **中通客车（000957.SZ，2022 客车龙头）**：2022/5/13 - 6/21 共 13 板，板块联动（汽车板块同期涨停 20+ 只）+ 龙虎榜共振（多家知名游资）。
3. **剑桥科技（603083.SH，2023 AI 算力龙头）**：2023/4-6 月 5 波行情，每波龙头地位未变——赵老哥"题材容量"典型。
4. **捷荣技术（002855.SZ，2023 华为产业链）**：2023/9 连板 11 板，板块效应强 + 机构游资共振。

**限制因素**：
- ① T+1 制度：涨停次日不能立即卖，**首板炸板 = 亏 10%+**（不是尾盘博弈的"小亏"）。
- ② 量化资金挤压：2022 年起量化高频对"二板定龙头"形成 30% 以上的胜率挤压，**需结合"机构净买入"过滤量化陷阱**。
- ③ 监管"特停"风险：连板 ≥ 8 板后常被交易所"特停"（停牌核查），**特停后复牌常常补跌 20%**。
- ④ 题材退潮快：单一题材活跃期 5-10 日，**节奏踩错一次亏 30%+**。
- ⑤ 适用窗口：只适合**题材市**（涨停家数 > 60 只/日、板块轮动快），震荡市和熊市题材龙头失败率 > 70%。

## 六、代表案例

### 浙江建投（002761.SZ，2022 基建二波）

- **首板识别（2/7）**：2022/2/7 基建板块首批涨停潮（当日板块涨停 8 只），浙江建投封单稳定（封单/流通市值 1.2%），龙虎榜买一为"宁波桑田路"知名游资。
- **二板确认（2/8）**：次日高开 6% 缩量涨停，二板定龙头（板块唯一二板）。
- **三板加仓（2/9）**：三板放量（量能 > 二板 50%），加 30% 仓。
- **首阴减仓（2/22）**：7 板后收阴 -7.2%，按纪律减仓 50%。
- **二波参与（2/23-2/24）**：回踩 10 日线 + 反包形态，二波持仓 30%，到 3/2 高点 41.7 元（首阴低点 19.95 元，109% 涨幅）。
- **核心信号**：板块联动 + 龙虎榜游资共振 + 题材（基建）持续催化（专项债加速）。

### 中通客车（000957.SZ，2022 客车 13 板）

- **首板（5/13）**：国常会"阶段性减征部分乘用车购置税 600 亿元"催化，板块 12 只涨停，中通客车是流通市值 30-50 亿区间的核心标的。
- **二板（5/16）**：周一高开 5% 缩量二板，板块联动 8 只涨停。
- **三板+（5/17-6/21）**：13 板全程未破 5 日线，期间"国务院政策持续 + 地方跟进"催化。
- **首阴（6/22）**：跌停 -10.01%，但次日未封跌停 + 收阳"假阴真阳"，题材未退潮。
- **二波（6/22-7/8）**：跌停 -10% 入 40% 仓，6/23 反包加 30%，6/24 突破 5 日线满仓，7/8 冲至 22.55 元（首阴 16.30 元起 38% 涨幅）。
- **核心信号**：政策催化延续 + 板块联动 + 龙头地位未被替代。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重      | 核心指标                          | 满分规则                                                       |
| -------- | --------- | --------------------------------- | -------------------------------------------------------------- |
| 基本面   | **7.5%**  | 题材稀缺性、龙头地位              | 唯一二板+板块龙头→100分，题材稀缺但非龙头→50分，普通题材→0分    |
| 估值     | **12%**   | 流通市值、PE 分位                 | 流通市值 30-200 亿+PE<行业 60%分位→100分，>200 亿或<30 亿→50 分 |
| 技术面   | **30.5%** | 均线多头、缩量加速、K线形态       | MA5>MA10>MA20+缩量涨停→100分，多头排列但放量→60分，破 5 日线→0分 |
| 情绪/题材 | **39%**  | 涨停基因、板块联动、龙虎榜        | ≥4 板+板块涨停>5 只+游资共振→100分，2 板+板块涨停 3-5 只→60 分 |
| 风险     | **11%**   | 监管风险、炸板率、退潮期          | 监管未特停+炸板率<20%+涨停家数>60→100分，特停风险→0分          |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

- 大盘处于 20 日均线下方且缩量
- 全市场涨停家数 < 20（退潮期不打板）
- 龙头地位被替代（板块龙头切换）
- 监管"特停"频繁（政策风险不可控）
- 个股流通市值 < 30 亿或 > 500 亿（炸板风险 / 流动性差）

### 调用方式

debate 模式中重点关注：龙虎榜买一席位（游资 vs 机构）、近 30 日涨停记录、板块联动家数、流通市值。**题材龙头特有过滤**：技术面分 ≥ 60 + 题材联动分 ≥ 60 才进候选池；任一不达标直接淘汰。

**代码示例：**

```python
from data import get_quote, get_kline
from common import to_float

quote = get_quote("sz002761")
bars = get_kline("sz002761", scale=240, datalen=30)

# 涨停基因（近 30 日）
limit_up_count = sum(1 for b in bars if to_float(b.get("pct_chg", 0)) >= 9.5)
gene_score = 100 if limit_up_count >= 4 else 60 if limit_up_count >= 1 else 0

# 流通市值
mkt_cap = to_float(quote.get("circulating_cap", 0))
if 30 <= mkt_cap <= 200:
    cap_score = 100
elif 10 <= mkt_cap <= 500:
    cap_score = 50
else:
    cap_score = 0

# 题材龙头特有过滤
if gene_score < 60 or cap_score < 60:
    print("观察池（题材龙头不过）")
else:
    print("题材龙头候选池")
```
