# 价值机构锚（value_institution）

> v2.4.0 起合并：价值双锚（巴菲特+段永平）+ 机构派（高瓴/红杉/淡马锡框架）。本质是用"四表一注"的可量化框架筛选，再用"商业模式+企业文化+行业空间+公司治理"的视角验证护城河厚度与长期确定性。
> 数据单位：财务指标百分比，案例金额以人民币（RMB）计。

## 一、核心哲学

价值机构锚是 A 股价值投资派的"三口径"决策者。巴菲特提供了"高 ROE + 低 PE + 强 FCF + 合理负债率"的可量化筛选四件套，段永平补充了"商业模式 + 企业文化 + 长期主义"的非量化护城河评估，机构派增加了"行业空间 + 竞争格局 + 公司治理 + 5-10 年产业投资"的长期确定性维度。三者一硬两软，互相校验：

- **数据维度（巴菲特口径）**：长期 ROE ≥ 15%、毛利率 ≥ 40%（消费/制药）、资产负债率 < 50%、自由现金流占净利润 > 80%。**核心命题**：找出"用最少资本创造最多利润"的公司。
- **文化维度（段永平口径）**：商业模式要"敢想 10 年后还赚钱"、企业文化以"消费者导向"为先、管理层"正直 + 平常心 + 长期主义"、股权结构稳定无关联交易风险。**核心命题**：找出"不靠风口也能持续赚钱"的公司。
- **产业维度（机构派口径）**：行业空间 TAM ≥ 1000 亿、竞争格局 CR3 ≥ 30%、管理层行业经验 ≥ 10 年、创始团队持股 ≥ 10%、关联交易 < 5%。**核心命题**：找出"大赛道中的确定性龙头"。
- **安全边际优先于择时**：即使是好生意，也要等"价格合理甚至低估"才出手——段永平名言"买股票就是买公司，但买公司不等于任何价格都买"。
- **不熟不做**：看不懂的商业模式不参与，**看不懂分两种——真不懂（不投）和假装懂（最危险）**。
- **与原 value_anchor / institution 的差异**：原 value_anchor 偏财务数据+文化叙事，原 institution 偏行业空间+治理结构；两者 90% 权重同构（基本面+估值+安全边际），实测评分仅差 0.5 分。合并后要求三者都达标才进入候选池，过滤掉"高 ROE 但行业天花板可见"和"好赛道但商业模式过时"和"好文化但财务失真"的三类陷阱。

## 二、选股标准（可量化）

| 维度     | 指标                | 阈值                              | 来源       |
| -------- | ------------------- | --------------------------------- | ---------- |
| 基本面   | 长期 ROE            | **≥ 15%** 连续 5 年               | 巴菲特口径 |
| 基本面   | 净利率              | **≥ 15%**                         | 巴菲特口径 |
| 基本面   | 毛利率              | **≥ 40%**（消费/制药）/≥ 30%（制造业） | 巴菲特口径 |
| 基本面   | 营收增速            | **≥ 20%**（连续 3 年）            | 机构派口径 |
| 估值     | PE 绝对值           | 历史 5 年分位 **< 40%**           | 双锚共识   |
| 估值     | PE/ROE              | **< 3** 极度低估，< 1 罕见        | 巴菲特口径 |
| 估值     | PEG                 | **≤ 1.5**                         | 机构派口径 |
| 估值     | 安全边际            | 买入价相对内在价值折扣 **≥ 30%**  | 巴菲特/段永平共识 |
| 财务安全 | 资产负债率          | **< 50%**（金融业除外）           | 巴菲特口径 |
| 财务安全 | 经营现金流/净利润   | **> 0.8** 持续 3 年               | 巴菲特口径 |
| 行业空间 | TAM（总可寻址市场） | **≥ 1000 亿元**（5 年后）         | 机构派口径 |
| 行业空间 | 行业增速            | **≥ 15%**（未来 5 年 CAGR）       | 机构派口径 |
| 竞争格局 | 市占率              | 行业前 3（最好第 1）              | 机构派口径 |
| 治理     | 商业模式            | 看得懂 + 10 年后还成立            | 段永平口径 |
| 治理     | 管理层              | 正直 + 长期主义 + 股东友好        | 段永平口径 |
| 治理     | 关联交易占比        | **< 5%**                          | 机构派口径 |
| 治理     | 商誉/净资产         | **< 30%**                         | 机构派口径 |

**护城河四要素**（巴菲特 1999 财富文章 + 段永平延伸）：① 品牌（茅台/片仔癀）；② 成本（GEICO/BNSF）；③ 网络效应（运通卡/微信）；④ 转换成本（苹果 iOS 生态/SAP 企业软件）。段永平补充第五条"**企业文化**"为最深护城河。机构派补充第六条"**行业壁垒**"（牌照/技术/规模效应）。

**机构派"三好"选股标准**（高瓴张磊《价值》总结）：
1. **好生意**：行业空间大 + 商业模式可持续 + 护城河深。
2. **好公司**：管理层优秀 + 公司治理透明 + 长期主义。
3. **好价格**：DCF 折现合理（机构派愿意持有 5-10 年，所以对短期估值容忍度高）。

## 三、买卖规则

- **买入条件**：三口径同时通过——量化指标（ROE/PE/FCF/负债率）+ 质化指标（商业模式/企业文化）+ 产业指标（行业空间/竞争格局/治理）都达标，且**当前价相对内在价值折让 ≥ 30%**。
- **分批建仓**：通过初筛后，分 3 批介入——**初筛入 30%（确认基本面）/回调 10% 加 30%（确认情绪）/跌破关键均线 5% 加 40%（极限仓位）**。**不在单一时点满仓**——价值投资最大的敌人是"机会成本焦虑"。
- **加仓触发**：股价下跌但**基本面与行业趋势未变**时加仓；**基本面恶化时（业绩失速、护城河被颠覆、管理层诚信问题）反而减仓**。
- **持有期**：默认"永远"（段永平名言"持有 10 年的股票和持有 10 分钟的股票，决策逻辑应该是一样的"），机构派视角下为 **5-10 年**，但下设 4 个卖出条件（见 §四）。

## 四、仓位与止损

- **集中持仓**：价值机构锚追求"重仓少数确定性最高的标的"，单一标的可达 **10%-25%** 总仓位。
- **组合上限**：前 3 大持仓 ≤ 50%；单一行业 ≤ 40%；前 5 大持仓 ≤ 70%。
- **不止损哲学**：与原双锚一致——**好生意不设价格止损，只设"基本面恶化"止损**。若价格下跌但基本面与行业趋势未变，加仓而非减仓。
- **4 类卖出条件**（巴菲特 1996/2014 + 段永平 2010/2018 + 机构派综合）：
  1. **估值严重高估**：PE 超过历史 5 年 90% 分位 且 内在价值折让 < 10%。
  2. **找到更便宜的替代品**：标的基本面仍好，但同业出现 PE/ROE 更优的标的。
  3. **基本面恶化**：ROE 连续 2 年下行、护城河被颠覆（技术替代/政策颠覆）、管理层诚信问题。
  4. **行业空间见顶**：TAM 增长停滞、竞争格局恶化（龙头被颠覆）。

## 五、A 股适用边界

**符合"价值机构锚"框架的 A 股代表**（2023-2024 数据）：

1. **贵州茅台（600519.SH）**：ROE 33%、毛利率 92%、净利率 53%、资产负债率 18%、关联交易 < 1%——三口径全维度超标；段永平本人重仓，机构持仓占比 30%+。
2. **宁德时代（300750.SZ）**：动力电池全球市占率 35% + 储能业务高速增长 + 管理层曾毓群深耕电池 30 年——高瓴 2020 年定增 100 亿入股，机构派典型。
3. **迈瑞医疗（300760.SZ）**：医疗器械国产替代 + 海外市场扩张 + 管理层李西廷深耕医疗器械 30 年——高瓴/红杉多次调研。
4. **长江电力（600900.SH）**：ROE 15.5%、经营性现金流 ≈ 净利润 1.6 倍、合同模式（"准债券"）——确定性收益标的，符合巴菲特"永远持有的好生意"定义。
5. **片仔癀（600436.SH）**：ROE 24%、毛利率 79%、国家保密配方构成"品牌+牌照"双重护城河。

**限制因素**：
- ① A 股信息披露透明度低于美股（关联交易、商誉减值调节空间大），三口径的"治理"维度需要额外做尽调。
- ② A 股退市机制弱，"永远持有"承担政策风险；2024 年新国九条强化退市后，限制有所缓解。
- ③ 散户结构（A 股成交 60% 来自散户成交 60%）放大波动，**价值机构锚标的在牛市可能被严重高估**，需要"不参与泡沫"的纪律。
- ④ 国企治理结构与民企不同：段永平口径"管理层是否股东友好"在国企不适用——改为"政策稳定性 + 行业垄断地位"。
- ⑤ 政策风险：行业政策变化（如双碳、医保集采）对长期持有有颠覆性影响，机构派在政策敏感行业的尽调需加倍。

**实操调整**：DCF 折现率取 **10%**（A 股波动率高于美股）；永续增长率取 **2-3%**（名义 GDP）；对国企需折扣 20% 治理风险溢价。

## 六、代表案例

### 贵州茅台（600519.SH，2013-2014 塑化剂危机）

- **三口径初筛（2013 年）**：ROE 38%、毛利率 92%、PE 9 倍、负债率 18%——财务数据"完美"；商业模式（高端白酒）+ 文化（千年品牌+护城河）"无可挑剔"；行业空间（高端白酒 TAM 2000 亿+）+ 竞争格局（龙头市占率 60%+）"确定性高"。
- **买入窗口（2013Q4-2014Q1）**：塑化剂事件 + 限制三公消费，PE 从 25 倍跌至 9 倍，跌幅 60%——**三口径判断"基本面未变+价格恐慌"**。
- **持有回报**：2014 年初 100 元 → 2023 年末 1720 元（含分红），**9 年 17 倍**，年化复合 33%。

### 宁德时代（300750.SZ，2020-2024 高瓴重仓）

- **深度尽调（高瓴 2018-2020）**：管理层访谈 50+ 次、行业专家访谈 30+ 次、客户访谈 20+ 次、供应商访谈 15+ 次。
- **三口径验证**：数据维度 ROE 20%+、毛利率 25%+、FCF 正；文化维度管理层长期主义+工程师文化；产业维度动力电池 TAM 5 万亿+行业增速 30%+龙头护城河。
- **持仓回报**：高瓴 2020 年 7 月 100 亿入股（161 元/股），2024 年中股价 200 元（+24%，考虑分红再投入），长期持有 4 年回报远高于短期博弈。

## 九、评分矩阵（debate 模式用）

### 维度权重

| 维度     | 权重    | 核心指标                              | 满分规则                                                       |
| -------- | ------- | ------------------------------------- | -------------------------------------------------------------- |
| 基本面   | **45%** | ROE、净利增速、毛利率、营收增速、行业空间 | ROE≥20%+营收增速≥20%→100分、≥15%→75分、≥10%→40分、<10%→0分    |
| 估值     | **22.5%** | PE、PE/ROE、PEG、安全边际            | PE分位<20%且PEG<1→100分，分位20-50%→60分，分位>80%→0分         |
| 技术面   | **5%**  | 长期趋势方向                          | 仅参考 250 日均线方向，上升→60 分，下降→0 分                    |
| 情绪     | **5%**  | 市场恐慌程度+机构持仓变化             | 极度恐慌→100 分，正常→50 分，亢奋→0 分                          |
| 安全边际 | **22.5%** | 负债率、FCF、护城河、治理、行业天花板 | 负债率<30% 且 FCF/净利>0.8 + 治理透明 + 行业天花板远→100 分     |

总分 = Σ(维度分×权重)，**≥70=强烈看多, 60-69=看多, 40-59=中性, 30-39=看空, <30=强烈看空**

### 一票否决

> v2.5.0 Phase 2 改造：原"一票否决->降至20"已升级为风险分级体系（刚性底线归零 + 弹性风险系数折扣）。以下条件由 `experts/veto_evaluator.py` 自动评估：
> - **刚性底线**（risk_coeff=0.0，归零）：商誉/净资产>50%、管理层欺诈
> - **弹性风险系数**（0.3-0.6，折扣但非零）：ROE<10%、FCF为负、PEG>2.5
> - **不可评估**（LLM 判断）：商业模式、行业天花板、管理层短视

- ROE < 10% 或负债率 > 70%（金融业除外）-> 弹性 risk_coeff=0.3-0.5
- FCF 连续 2 年为负 -> 弹性 risk_coeff=0.3-0.6
- 公司涉财务造假或管理层失信 -> 刚性归零
- 商业模式看不懂（"不熟不做"）-> 需定性判断，LLM 评估
- 行业空间天花板可见（5 年 TAM 增长 < 5%）-> 需行业数据，LLM 评估
- 管理层短视（季度业绩导向、不做长期投入）-> 需定性判断，LLM 评估

> **模型边界**：对强周期资产，本体系盈利能力评估存在结构性局限（单期增速无法区分"周期顶部高增速"与"成长性高增速"）。v2.5.0 Phase 3 新增 `factors/cyclical.py` 周期矩阵 + DCF 三情景估值缓解此问题，但低分仍应解读为"不符合本体系标准"而非"价格将下跌"。

### 调用方式

debate 模式中，从 `scripts/data/` 获取最新 ROE/PE/负债率/FCF/营收增速/行业数据后，按上表打分。**价值机构锚特有过滤**：估值分 ≥ 60 分 且 基本面分 ≥ 60 分 才进入"候选池"；任一维度不达标直接降为"观察"。

**代码示例：**

```python
from data import get_quote, get_finance
from common import to_float

quote = get_quote("sh600519")
fin = get_finance("sh600519")[0]

# 三口径初筛：财务 + 估值 + 产业必须都过
roe = to_float(fin.get("ROEJQ", 0))
pe = to_float(quote.get("pe", 0))
rev_growth = to_float(fin.get("TOTALOPERATEREVETZ", 0))

# 基本面维度
if roe >= 20 and rev_growth >= 20:
    base_score = 100
elif roe >= 15:
    base_score = 75
elif roe >= 10:
    base_score = 40
else:
    base_score = 0

# 估值维度
if pe <= 15:
    val_score = 100
elif pe <= 25:
    val_score = 60
elif pe <= 40:
    val_score = 25
else:
    val_score = 0

# 三口径过滤：任一不达标不进候选池
if base_score < 60 or val_score < 60:
    print("观察池（价值机构锚不过）")
else:
    print("价值机构锚候选池")
```

### 发言约束

debate 模式中本专家的 `reason` 核心理由字段须遵守以下规则（`experts/formatter.py` 会在渲染层检测并标记违规）：

- **否决条件**：调用一票否决（如"ROE<10% 否决"）必须引用实际财务数值（如"ROE 8.2%<10%->触发否决"），不得仅凭定性印象。
- **安全边际**：陈述安全边际评分时须展示计算依据（如"FCF/股 2.1 元 + 负债率 28%<30%->安全边际 90"），不得省略为"安全边际充足"。
- **维度拆分**：展示 breakdown 时须用三元组格式 `原始分 × 权重 = 贡献值`（如"估值：33 × 22.5% = 7.42"）。
- **模型边界**：对强周期股得低分时，应说明"不符合价值机构锚的投资标准"（如 ROE 波动大/FCF 不稳定），不得推导为"反向指标"或"价格将下跌"。
