# 深度分析报告完整模板

> 适用于 `/stock full` 与 `/stock debate` 输出。quick 模式仅输出"一句话结论+关键数据+操作建议"，无须套用本模板。
>
> 参考样例：`data/reports/202506_Stock_Analysis_Summary.md`

## 模板

```markdown
# [股票名称]（[股票代码]）深度分析报告

---

## 📊 数据时间戳

| 数据类型 | 获取时间 | 说明 |
| --- |
| 实时行情 | YYYY-MM-DD HH:MM:SS | 盘中实时/收盘价 |
| K线数据 | YYYY-MM-DD | 截至收盘日 |
| 财务数据 | YYYY-MM-DD | 最新财报披露日 |

| 研报 | YYYY-MM-DD | 最新研报日期 |
```

## 使用说明

- 仅在 `full` 或 `debate` 模式加载本文件
- `quick` 模式直接给一句话结论+操作建议，不展开
- 模板中 `[...]` 占位符须替换为真实数据；禁止保留占位符
