# 通用防护规则（v2 扩写）

> 所有 skill 共用，按需加载。
> 本文件是生成器护栏的**权威源**，不在用户报告中渲染。报告渲染时仅展示结论（🛡️ 数据护栏条，见对应 skill 的输出模板页脚）。
> 改动本文件后，请同步更新被引用此文件的所有 SKILL.md 与各 skill 的 Step 3 章节。

## 一、数据真实性

- 报告中涉及的日期、金额、事件等事实性信息，**必须有脚本输出佐证**。
- 脚本未返回的数据不得编造，缺失项标注为"未覆盖"。
- 不要虚构实时价格、最新公告、研报评级或成交数据。
- 数据源失败时说明失败的数据源和影响，仍可用已有数据给低置信度结论。

## 二、事件日历

- 涉及个股分析的 skill，必须调用 `events.py` 获取事件日历。
- 未调用 `events.py` 时，不得在报告中提及任何事件日期（财报、分红、解禁等）。
- 事件信息必须以脚本输出为准，不凭记忆推断。

## 三、合规措辞（A 股非持牌投顾约束）

> 本工具非证券投资咨询业务持牌机构。任何"投资建议"表述触发合规红线。措辞统一规则：

**禁用措辞**（替换）：
- "建议买入 / 卖出 / 加仓 / 减仓 / 清仓"
- "目标价 XX 元"（无依据时）
- "止损位 / 止盈位"（用"观察下沿 / 观察上沿"）
- "明日高开 / 今日尾盘"（任何隔日预判）
- "强烈推荐 / 强推"（配合"建议"动词时）

**替代措辞**：
- 研判 / 观察 / 倾向 / 参考仓位 / 观察上沿 / 观察下沿
- "综合研判：偏多，置信度 X%"  ← 用"研判"不用"建议"
- "操作观察：+X.XX 突破 / -X.XX 跌破"  ← 用"观察"不用"买入"
- "结论等级：可参考 / 一般 / 不确定"  ← 不用"强推/推荐"

**免责声明三处冗余**：
1. 决策卡片下方 1 行斜体（读完倾向立刻提醒）
2. 数据护栏条内 1 行（合规自证位置）
3. 页脚完整段（正式承担位置）

> A 股合规硬底线：发布"投资建议"需证券投资咨询业务资格。非持牌机构对外输出"买入/卖出"构成合规风险。

## 四、报告质量约束（原审查 #N 集中）

| # | 规则 | 适用范围 |
| --- | --- | --- |
| 去重 | 同一数据点 ≤2 次（数据表 1 次 + 分析论证 1 次），禁止 3 次以上重复 | 所有章节 |
| 评分统一 | `A+/A/B+/B/C` 五档字母制，禁用 ⭐ 任何符号 | 所有评分处 |
| 表格合并 | 营收/净利/盈利三表合并为"核心财务数据表"，其他章节只引用增量信息 | full / debate 模式 |
| 排名限定 | 仅限本行业（申万二级），禁止跨行业排名（vs 茅台/长电/神华） | 所有排名处 |
| 关键数字单一信源 | 价位/分位/前低/上轨在 30 秒研判段统一定义；下文用语义引用，禁原始数字 3+ 次重复 | full / debate 模式 |
| 数据来源 | 核心数据表每行须标来源（年报/季报/预增公告/研报 API），禁止推算值不标来源 | full / debate 模式 |
| 增速分列 | 利润增速 vs 营收增速，必须分别标注，禁止笼统称">X%" | full 模式 |
| 杜邦对账 | 重建 ROE 与原始 ROE 对账偏差 >2pp 必须标注 ⚠️ 并核查数据来源 | full 模式 |
| ROE 口径 | 必须明示"全年 / 季报 / 年化"口径；混合表达禁止 | full 模式 |
| 情景概率标注 | 必须标注"主观情景假设，非统计预测"；禁伪装为统计预测 | full / debate 模式 |
| 降级字段跟踪锚 | 字段降级时附 1-2 句用户自跟踪锚点，禁仅标"未覆盖" | full / debate 模式 |
| PE 多口径 | 动态 PE 多口径并列（Q1 年化 / H1 预增修正 / 机构全年预测），单一口径无效 | full 模式 |
| 目标 PE | 目标 PE 必须有论证依据（可比公司 / 机构隐含 PE / PEG），禁止"目标 PE 10×"无依据表述 | full 模式 |
| 业绩预告 | 必须从公告原文提取，禁止 ×2 年化（Q1 占 H1 比例 ≠ 50% 应按实际比例外推） | full / debate 模式 |
| 关联交易 | 必须基于公告事实（收购标的/评估增值率/年化收入/占净资产比/对价公允性），禁止"控股股东注入利好"无依据猜测 | full 模式 |
| 解禁分析 | 禁止将股价波动简单归因于解禁；必须对比同期大盘/板块涨跌 + 量化解禁冲击（可解锁股数/流通股本） | full 模式 |
| 风险维度 | 限定（行业周期 / 原料价格 / 产能投产 / 环保政策 / 汇率 / 流动性），禁止预判隔日涨跌 | 所有风险表述 |
| 专家 reason | 必须引用触发评分矩阵的具体数据值（如 PE 分位 85%），禁止纯定性结论 | debate 模式 |
| 禁用表述 | reason 禁止使用"反向加分 / 反向指标 / 该分数可作为反向参考"等表述 | debate 模式 |
| 模型边界 | 承认本体系对强周期资产盈利能力评估的结构性局限；低分仅代表"不符合本体系投资标准"，不构成反向预测 | debate 模式 |

> 渲染层兜底：`experts/formatter.py::_validate_reason` 在输出时检测 reason
> 是否含数据引用、是否出现禁用表述，缺失时追加 `⚠理由缺数据引用`
> 或 `⚠含禁用表述` 警告标记。

## 五、章节路由（按股票类型条件渲染）

| 类型 | 保留章节 | 删除章节 |
| --- | --- | --- |
| 消费/医药 | 大盘状态 / 板块强度 / 北向偏好 / 分产品收入结构 | 宏观桥（原油/碳酸锂）/ 组合相关性 / 题材轮动 |
| 科技/成长 | 大盘状态 / 行业 beta（一行）/ 流动性比率 / 杠杆反身性 | 组合相关性 / 分产品收入结构 / 宏观桥 |
| 周期（有色/化工/能源） | 宏观桥（油/锂/黄金）/ 板块强度 / 情绪周期 / 产能扩张 | 组合相关性 / 北向偏好 / 分产品 |
| 公用事业 | 大盘状态 / 板块强度 / 分产品收入结构 | 全部择时与宏观字段 |

> 北向资金 2024 年起监管限制披露，盘后数据延迟，**全类型降级为附录/可选**。
> 组合相关性属于组合层指标，与个股决策无关，**移出个股报告**（仅 `/portfolio` 健康检查保留）。
> IF/IC/IH 期货基差属于股指期货套利指标，目标用户错位，**移出个股报告**。
> 题材轮动属于当日择时信号，长线投资者无需，**移出个股报告**。

## 六、降级与失败处理

- 字段拉取失败 → 标注 ⚠️降级，不静默省略
- 板块数据失败 → RPS vs 块对比不可用，不渲染空表
- 大盘失败 → regime 默认 `defensive`，标注降级原因
- yfinance 失败 → fixture fallback，时点标注
- 任一字段失败不阻塞主流程，但不可向下游隐藏失败
