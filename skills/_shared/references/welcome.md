# 🎯 欢迎使用 stock-analyzer-skill

## 这是什么？

把它装进 Claude Code，你就有了 **9 位常驻的 A 股投资分析师**。

> 你不需要写代码、不需要懂量化、不需要任何 API 密钥。
> 你只需要在 Claude Code 对话框里打 `/stock 贵州茅台`，3 分钟拿到 5 层专业分析。

## 🚀 3 步 30 秒上手

```text
1. /screener init           初始化股票池（首次需要，30 秒，零配置；网络差用 /screener init default 走预置数据）
2. /stock sh600519 quick    分析一只股票（基本面 + 估值 + 技术面，3 分钟；用代码最稳，名称匹配可能错配）
3. /market quick            看今天市场涨跌（30 秒）
```

跑完这 3 步，你就掌握了 80% 的日常用法。

> 💡 **跳过 init 也能跑**——系统会在第一次用 `/stock` 时自动提示。

## ❓ 第一次用会遇到的问题

**Q：股票池不初始化会怎样？**
A：第一次跑 `/stock` 时系统会自动提示你跑 `/screener init`，不强制。

**Q：能离线用吗？**
A：可以。`/screener init default` 走预置数据，零网络请求。

**Q：数据源挂了怎么办？**
A：内置 28 个数据源（腾讯/东财/新浪/雪球/同花顺/...）+ 熔断器，单源失败自动切下家。

**Q：分析结果能直接拿去交易吗？**
A：❌ 不能。所有输出仅供研究框架参考，不构成投资建议。
