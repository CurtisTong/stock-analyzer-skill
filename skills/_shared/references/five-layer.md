# 五层分析框架

> 适用于 /stock、/financial-analyst、/investment-researcher。详细阈值见包根目录 `methodology.md`。

## 五层定义

| 层级       | 核心问题           | 关键指标                                    | 输出评级 |
| ---------- | ------------------ | ------------------------------------------- | -------- |
| 基本面     | 公司赚不赚钱？     | ROE、营收/净利增速、毛利率、负债率、现金流  | A+/A/B/C |
| 估值       | 买得贵不贵？       | PE、PB、PEG、PE/ROE                         | 低估/合理/偏贵 |
| 技术面     | 什么时候买？       | 均线、MACD、KDJ、BOLL、量能、形态           | 强/中/弱 |
| 板块       | 风口在不在？       | 行业景气、轮动位置、ETF 资金流             | 启动/主升/高潮/退潮/弱势 |
| 风险收益比 | 期望值多少？仓位？ | 情景分析（牛/基/悲）、凯利公式              | 期望收益% |

## 评级阈值

| 评级 | 基本面                  | 估值                 | 技术面           |
| ---- | ----------------------- | -------------------- | ---------------- |
| A+/Buy | ROE>20% 且 增速>50%   | PE/ROE<1 且 PEG<0.5  | 评分>65, 多重买入 |
| A/Buy  | ROE>15% 且 增速>20%   | PE/ROE<3 且 PEG<1    | 评分>60, 买入信号 |
| B/Hold | ROE>10% 且 增速>0     | PE/ROE<5             | 评分 50-60, 中性  |
| C/Sell | ROE<10% 或 增速<0     | PE/ROE>5 或 PE>30    | 评分<50, 卖出信号 |

## 上下游交接字段

每个 skill 输出给下游时，保留：

- `fundamental_rating` (A+/A/B/C)
- `valuation_view` (低估/合理/偏贵)
- `sector_context` (板块景气+轮动位置)
- `technical_trigger` (买入触发+支撑阻力+失效位)
- `position_plan` (试探/标准/重仓/回避)
- `confidence` (高/中/低 + 数据缺口说明)
- `thesis_breaker` (论点破灭条件)
