# 统一输出模板

> 12 个 skill 共用，按需加载。`common/formatters.py` 提供程序化支持。

## 结构

每个 skill 输出必须遵循以下结构：

```
🎯 一句话结论

[详细分析内容...]

────────────────────────────────────
📊 数据时间戳: YYYY-MM-DD HH:MM | 数据源: 腾讯行情, 东方财富财务 | ⚠️ 失败源: 新浪K线
```

## 规范

| 位置     | 内容                                | 说明                                              |
| -------- | ----------------------------------- | ------------------------------------------------- |
| **首行** | `{emoji} {conclusion}`              | emoji 可选：🎯 默认 / 🟢 看多 / 🔴 看空 / 🟡 中性 |
| **中间** | 详细分析                            | 表格、要点、评分等，按各 skill 自身格式           |
| **尾行** | `📊 数据时间戳 \| 数据源 \| 失败源` | 缺失项可省略，但时间戳必须有                      |

## Emoji 约定

- 🎯 默认（操作建议、中性结论）
- 🟢 看多（买入、加仓、强势）
- 🔴 看空（回避、减仓、弱势）
- 🟡 中性（观望、持有、不确定）
- ⚠️ 警告（风险提示、数据缺失）

## quick 模式简化

quick 模式可省略详细内容，仅保留首行结论 + 尾行数据源。
