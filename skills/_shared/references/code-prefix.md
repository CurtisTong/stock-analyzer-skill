# 股票代码前缀约定

> 12 个 skill 共用，按需加载。

## 大小写规则

| 用途     | 前缀格式 | 样例              | 说明                            |
| -------- | -------- | ----------------- | ------------------------------- |
| 行情/报价 | 小写 `sh`/`sz` | `sh600989`, `sz000807` | 腾讯 qt.gtimg.cn 行情接口 |
| K 线     | 小写 `sh`/`sz` | `sh600989`        | 新浪 K 线接口                   |
| 财务     | 大写 `SH`/`SZ` | `SH600989`        | 东方财富 PC_HSF10 接口          |
| 公告/研报 | 纯数字         | `600989`          | 东方财富 np-anotice 接口        |

## 输入容错

- `sh600989` / `SH600989` / `600989` / `600989.SH` 全部视为同一只股票
- 股票名称（如"贵州茅台"）按名称模糊匹配内置代码表
- 沪市代码以 `6` 开头（除 `5` 开头 ETF/基金），深市以 `0`/`3` 开头

## 板块后缀

- 北交所代码以 `8`/`4` 开头
- 创业板以 `300`/`301` 开头
- 科创板以 `688` 开头
- ETF 以 `510`/`511`/`512`/`513`/`515`/`516`/`588` 开头

## 跨市场代码（v1.15.0+）

需要安装 `yfinance` 包（`pip install yfinance>=0.2`），未安装时自动跳过不影响 A 股。

| 市场   | 前缀格式 | 样例                | 说明                          |
| ------ | -------- | ------------------- | ----------------------------- |
| 美股   | `us:`    | `us:^gspc`, `us:spy` | yfinance 符号，保留 `^` 等特殊字符 |
| 港股   | `hk:`    | `hk:0700`, `hk:9988` | 数字代码，自动补齐 4 位 + `.HK` 后缀 |

- 美股支持指数（`us:^gspc` 标普500、`us:^ixic` 纳斯达克、`us:^dji` 道琼斯、`us:^vix` VIX）
- 港股支持中文名映射（腾讯、阿里、美团、小米、港交所等）
- 跨市场代码大小写无关：`US:SPY` 等价于 `us:spy`
