# Stock Analyzer Skill

A 股投资的 12 位分析师，常驻你的 Claude Code。

## 核心能力

- **五层分析框架**：基本面 · 估值 · 技术面 · 板块 · 风险收益比
- **8 人活跃专家圆桌（含 16 份专家人设）**：长线 5 人（林奇 · 索罗斯 · 价值机构锚 · 行业专家 · 风控）+ 短线 3 人（题材龙头 · 情绪技术 · 动量派）；另有 8 份 legacy 档案（巴菲特/段永平/徐翔/赵老哥/养家/作手新一/机构派/价值锚）保留研究
- **27 个数据源故障转移**：腾讯 · 东财 · 新浪 · 雪球 · 同花顺 · AkShare · efinance
- **零配置开箱即用**：内置预置默认股票池，无 token 即可启动

## 快速开始

```bash
# 1. 初始化股票池
/stock-init

# 2. 看大盘
/market quick

# 3. 分析个股
/stock sh600519 quick
```
