# Stock Analyzer Skill

A 股投资的 12 位分析师，常驻你的 Claude Code。

## 核心能力

- **五层分析框架**：基本面 · 估值 · 技术面 · 板块 · 风险收益比
- **9 人活跃专家圆桌（含 15 份专家人设）**：长线 4 人（巴菲特 · 林奇 · 索罗斯 · 段永平）+ 短线 4 人（徐翔 · 赵老哥 · 养家 · 作手新一）+ 7 active（价值双锚 · 题材龙头 · 情绪技术 · 行业专家 · 机构派 · 风控 · 动量派）
- **27 个数据源故障转移**：腾讯 · 东财 · 新浪 · 雪球 · 同花顺 · AkShare · efinance
- **零配置开箱即用**：内置预置默认股票池，无 token 即可启动

## 快速开始

```bash
# 1. 初始化股票池
/stock-init

# 2. 看大盘
/market quick

# 3. 分析个股
/stock sh600519 quick
```
