# stock-analyzer-skill v1.15.0 Release Notes

> 发布日期：2026-07-09
> Tag: `v1.15.0`
> 主题：**多轮审查修复 + 文档同步 + 数据域补齐**

本次发布为五轮深度审查的收口版本，重点是**质量债清理 + 工程化加固**，不引入破坏性 API 变更。

---

## 亮点

- **5 轮深度审查收口**：修复 12 个 P0 + 20+ 个 P1 + 30+ 个 P2 问题
- **专家系统 v2.4.0 收敛**：`value_anchor + institution` 合并为 `value_institution`；legacy 8 人降为研究档案
- **市场环境锚定 10 大维度**：multi-timeframe + 宏观-估值桥 + 杠杆-反身性 + 流动性+波动率 + 情绪周期 + 行业 beta + 组合相关性 + 题材轮动 + 北向资金
- **Phase 4 用户保护三重防线**：AI 免责 + 中文名解析 + 边界声明
- **数据正确性 hotfix 12 项**：volume/amount 双重归一化 / ST 检测 / 中枢合并 / 涨跌停识别 / tushare 代码映射等

## 主要改动

### Added（详见 CHANGELOG.md）

- **portfolio**：Brinson attribution + VaR/CVaR + oplog + undo
- **backtest**：50+ 只外样本回测 + 沪深 300/中证 500 基准
- **experts**：value_institution 合并 + buffett veto 转 warning + 维度名别名归一化
- **monitor**：6 个通知规则从 yaml 配置加载
- **portfolio-web**：Origin 白名单 + IP 限流防 CSRF
- **scripts**：markdown_table / numbered_table formatters + TTL jitter
- **validators**：NAME_TO_CODE 扩到 50+ 只股票（5 板块）
- **港股跨市场支持** + 盘前简报增强

### Fixed（详见 CHANGELOG.md）

- 数据正确性：volume/amount 双重归一化、tushare 北交所代码、ST 子串匹配、20cm 涨跌停检测
- 校准：legacy→active 名迁移修复静默数据丢失；中枢合并用并集
- 安全：SSRF/scheme 白名单 + CSV 路径注入防护
- 工程化：file IO 原子性、Python `from config` 导入路径、CB half_open 阈值守卫
- experts：维度 key 对齐 + P2-11 profile 回退 proxy + 短线均分驱动

### Changed

- **experts**：legacy 8 人（巴菲特/段永平/徐翔/赵老哥/养家/作手新一/value_anchor/institution）标记 `active: false`，保留为研究档案；active 圆桌稳定在 8 人 = 5 长线 + 3 短线
- **screener/market**：brief 默认输出；--full 切换详情
- **monitor**：alert direction 修正（target_sell ≤ ts = 触发止损）

## 数据域扩张

`fetchers/` 模块从 26 个扩展到 **27 个模块 / 35 个 fetcher 类**，新增：
- `chip` 域：MarginFetcher / HolderFetcher / TopHolderFetcher（覆盖融资融券 + 股东户数 + 十大流通）
- `event` 域：5 个 fetcher（财报/解禁/分红/增减持/违规）
- `performance_forecast.py`：业绩预告 fetcher（显式路径导入）

## 升级指引

无需任何代码改动；现有调用方式完全兼容。

如本地已安装旧版 skill，重跑 `./install.sh` 即可升级。
