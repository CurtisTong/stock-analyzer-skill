# stock-analyzer-skill v1.3.2 Release Notes

> 发布日期：2026-06-10
> Tag: `v1.3.2` · Commit: `b9b5ea4`
> 主题：**Skill 元数据标准化 + 共享 references + CI 测试防护**

本次发布是 12 个 skill 的工程化重构，不引入新功能，专注于**可发现性、可维护性、可测试性**三方面的质量提升。

---

## 亮点

- **12 个 skill 全面升级 frontmatter**：新增 `model` / `allowed-tools` / `version` 字段；3 个命令式 skill 加 `disable-model-invocation: true`
- **共享约定集中化**：3 份共享 references（代码前缀、脚本目录、五层框架）按需加载，消除 6 处重复段
- **172 个 CI 测试防护**：frontmatter 一致性 + cross-reference + install.sh 集成测试
- **GitHub Actions 集成**：PR 必跑 172 个测试，main 分支额外跑 install.sh

## 改动详情

### Added（新增）

- `skills/_shared/references/code-prefix.md` — 股票代码 `sh`/`sz`/`SH`/`SZ` 大小写约定
- `skills/_shared/references/script-catalog.md` — 脚本目录与参数
- `skills/_shared/references/five-layer.md` — 五层分析框架与评级阈值
- `skills/stock/reports/full-template.md` — stock 深度报告完整模板（外移 80+ 行）
- `skills/help/references/skill-catalog.md` — 12 个 skill 命令清单
- `tests/test_skill_metadata.py` — 100 个 SKILL.md 元数据校验
- `tests/test_skill_consistency.py` — 72 个 cross-reference + 关键词测试
- `tests/integration/test_install.sh` — install.sh 集成测试
- `.github/workflows/ci.yml` — GitHub Actions（Python 3.11/3.12 + install.sh）

### Changed（变更）

- 12 个 SKILL.md 全面升级 frontmatter（version / model / allowed-tools / disable-model-invocation）
- description 从硬编码 `/X 触发` 升级为能力 + 场景描述（平均 110 → 116 字符）
- 删除 5 处过期路径表述 `../../..`，统一为"Claude Code 工作目录即为项目根"
- 抽取 stock/market/sector/portfolio/financial-analyst/investment-researcher 的"数据获取"段为引用
- help SKILL.md 从 256 行瘦身到 82 行（拆出命令清单子文件）
- stock SKILL.md 从 246 行瘦身到 161 行（拆出报告模板子文件）
- `install.sh` 全局同步从 `cp -r` 改为 `ln -s`（单源真相）
- allowed-tools 改用通配符 `Bash(python3 scripts/*)` + `Read(**)` 放宽权限
- README/CLAUDE.md/plugin.json/marketplace.json 数字从 8/11 混用统一为 12
- `plugin.json` / `marketplace.json` version 从 1.0.0/1.1.0 升到 1.3.1

### Fixed（修复）

- `data/portfolio.json` 等运行时生成文件被引入 cross-reference 测试白名单
- 测试条件"硬编码 `/X` 命令"误判路径片段（修复正则为仅检测触发句）

## 升级指引

无需任何代码改动；现有 `python3 scripts/quote.py` 调用方式完全兼容。

如本地已安装旧版 skill（`~/.claude/skills/` 下有目录），重跑 `./install.sh` 即可升级为软链形式：

```bash
./install.sh
```
