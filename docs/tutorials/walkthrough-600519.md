## 常见问题

**Q: 数据源挂了怎么办？**
A: 自动故障转移，27 个 fetcher 依次尝试。用 `python3 scripts/monitor.py --sources` 查看状态。

**Q: 专家投票冲突怎么办？**
A: 由 `decide.md` 定义的整合规则裁决——加权投票 + 长短线分仓建议。

**Q: 回测胜率可信吗？**
A: 回测基于历史数据，不代表未来表现。用 `--with-backtest` 附加胜率作为参考，不要作为唯一决策依据。
