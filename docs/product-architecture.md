## 一、产品定位

### 1.1 产品概述

**stock-analyzer-skill** 是一款面向 A 股市场的智能股票分析工具，基于 Claude Code 平台运行。通过自然语言交互，为投资者提供基本面分析、技术面分析、选股筛选、仓位管理等全维度投资辅助。

### 1.2 核心价值

| 价值主张 | 描述 |
|----------|------|
| **智能化** | AI 驱动的多因子选股，减少主观判断 |
| **实时性** | 多数据源自动切换，秒级响应行情 |
| **专业性** | 融合缠论、8 人专家圆桌等机构级分析方法 |
| **零门槛** | 无需编程，通过自然语言即可使用 |

### 1.3 目标用户

| 用户类型 | 需求场景 |
|----------|----------|
| **个人投资者** | 选股参考、仓位管理、风险提示 |
| **量化爱好者** | 策略回测、因子研究 |
| **理财师** | 客户持仓分析、投资建议生成 |

### 2.2 功能详情

#### 2.2.1 行情查询 (`/stock`)

| 功能 | 描述 |
|------|------|
| 实时行情 | 多数据源自动故障切换 |
| 批量查询 | 最多支持 15 只/批 |
| 财务数据 | ROE/毛利率/负债率等核心指标 |
| 历史K线 | 5/15/30/240 分钟周期 |

#### 2.2.2 选股筛选 (`/screener`)

| 策略 | 说明 |
|------|------|
| `balanced` | 均衡策略 |
| `quality_value` | 质量价值 |
| `growth_momentum` | 成长动量 |
| `defensive` | 防御策略 |
| `turning_point` | 转折点策略 |

#### 2.2.3 技术分析 (`/technical`)

| 指标 | 功能 |
|------|------|
| MACD | 趋势动量 |
| KDJ | 短线拐点 |
| RSI | 超买超卖 |
| BOLL | 波动率 |
| MA | 均线系统 |
| 缠论 | 分型/笔/线段/中枢/背驰 |

#### 2.2.4 持仓管理 (`/portfolio`)

| 功能 | 描述 |
|------|------|
| 持仓记录 | JSON 本地存储 |
| 盈亏分析 | 持仓收益率 |
| 仓位建议 | 基于专家圆桌的综合评分 |

## 三、用户交互流程

### 3.1 典型使用场景

```
用户输入                          系统响应
─────────────────────────────────────────────────────
/stock sh600989        →    ┌────────────────────────┐
                            │ 宝丰能源 (600989)       │
                            │ 现价: 22.68 (-3.9%)     │
                            │ PE: 13.23 | ROE: 7.28%  │
                            │ 综合评分: 68分 → 偏多   │
                            └────────────────────────┘

/screener 消费         →    ┌────────────────────────┐
                            │ 入选: 12 只             │
                            │ 1. 贵州茅台  85分       │
                            │ 2. 五粮液    82分       │
                            │ ...                     │
                            └────────────────────────┘

/monitor               →    ┌────────────────────────┐
                            │ 数据源: 全部正常        │
                            │ 缓存: 928 个文件        │
                            │ 告警: 无                │
                            └────────────────────────┘
```

### 3.2 投资决策流程

```
1. 输入股票代码
       ↓
2. 数据获取 (多源自动切换)
       ↓
3. 五层分析 (基本面→估值→技术→板块→风险)
       ↓
4. 专家圆桌 (8 位专家投票)
       ↓
5. 仓位建议 (凯利公式 + 阈值控制)
       ↓
6. 输出报告 (评分/方向/风险/操作)
```

## 五、竞争优势

### 5.1 竞品对比

| 维度 | 本产品 | 传统券商App | 同类AI工具 |
|------|--------|-------------|------------|
| 数据源 | 7+ 免费源 | 仅自有 | 单一 |
| 分析深度 | 缠论+专家圆桌 | 基础指标 | 无 |
| 定制化 | 5 种选股策略 | 固定模板 | 弱 |
| 成本 | **免费** | 付费订阅 | 付费 |

### 5.2 核心差异点

1. **零依赖**: 核心脚本纯 stdlib，���需安装
2. **多源容错**: 单源失败自动切换
3. **行业差异化**: 30+ 行业阈值配置
4. **专家系统**: 8 位投资专家投票机制
