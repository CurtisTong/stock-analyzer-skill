## 五层分析框架

### 第 1 层：基本面筛选

> ⚠️ **重要**：以下阈值为默认值，实际筛选时按行业差异化执行。

- ROE > 15%（优秀）, > 20%（顶级）
- 净利增速 > 20%（成长）, > 50%（高速）
- 毛利率 > 30%（有壁垒）
- 负债率 < 60%（健康）
- 经营现金流/EPS > 1（利润含金量高）

#### 行业差异化阈值

| 行业类别 | ROE 优秀 | PE 上限 | 毛利率底线 | 负债率上限 |
| --- | --- | --- |
| 默认     | 20%      | 30      | 20%        | 70%        |
| 银行     | 12%      | 10      | 30%        | 92%        |
| 白酒     | 28%      | 40      | 50%        | 50%        |
| 医药     | 20%      | 35      | 40%        | 55%        |
| 科技     | 18%      | 50      | 25%        | 70%        |
| 半导体   | 18%      | 60      | 30%        | 65%        |
| 房地产   | 12%      | 12      | 20%        | 85%        |
| 钢铁     | 12%      | 12      | 10%        | 70%        |

> 完整行业阈值配置见 `data/industry_thresholds.json`

### 第 2 层：估值评估

- PE 绝对值 vs 行业对比
- PEG = PE / 净利增速（<1 低估, 1-2 合理, >2 偏贵）
- PE/ROE（<3 为好）
- 历史估值分位

### 第 3 层：技术面确认

- 30 日 K 线趋势（上升/横盘/下降）
- 关键支撑/阻力位
- 成交量变化（放量/缩量）
- 5 分钟分时形态（出货/吸筹/震荡）

### 第 4 层：板块与风格分析

- 板块轮动节奏
- 大小盘分化程度
- 市场风格（成长 vs 价值、进攻 vs 防御）
- 资金流向

### 第 5 层：风险收益比计算

- 情景分析（牛市/基准/震荡/悲观/极端）
- 概率加权期望收益
- 凯利公式：f = p - (1-p)/b
- 止损/止盈位设定

## 仓位管理

### 凯利公式

```
最优仓位 f = p - (1-p)/b
p = 胜率, b = 赔率(期望收益/最大风险)
```

**A 股适用性说明**：

- A 股涨跌停制度导致收益分布非正态（厚尾+截断），凯利公式假设的连续收益分布不完全成立
- 胜率 p 和赔率 b 在高波动环境下估计误差大，需要大样本验证
- **建议使用 1/3 Kelly**（而非半凯利），以仓位分级表为主要依据
- 仓位分级表（3%-15%）比 Kelly 公式输出更实用，Kelly 仅作参考上限

### 仓位分级

| 仓位   | 含义   | 适用场景              |
| ------ | ------ | --------------------- |
| 0%     | 不碰   | 基本面差/估值过高     |
| 3%     | 试探仓 | 等回调/方向不明       |
| 5%     | 标准仓 | 确认买入信号          |
| 8%     | 重仓   | 强烈看好+低位         |
| 10-15% | 核心仓 | 最强标的+安全边际充足 |

### 止损铁律

- 个股：跌破关键支撑位收盘确认即止损
- 组合：单日亏损>3% 减仓
- 板块：板块趋势转空减仓

### 集中度控制（C1）

| 限制类型 | 上限   | 说明                          |
| --- |
| 单只股票 | 20%    | 避免黑天鹅风险集中            |
| 同行业   | 40%    | 行业分散，规避行业系统性风险  |
| 同风格   | 60%    | 成长/价值风格平衡             |

### 时间止损（C2）

| 持有天数 | 无正收益      | 动作           |
| --- |
| 20 日    | 触发减半仓    | 减仓 50%       |
| 40 日    | 清仓          | 全部卖出       |

### 极端行情预案

| 条件                    | 动作           |
| --- | --- |
| 大盘单日跌幅 > 5%      | 全面减仓至 30% |
| 连续 3 日跌幅 > 3%/日  | 清仓           |
| 涨停家数 > 100 且持续  | 警惕泡沫，减仓 |

### 加仓规则（C4）

| 条件                          | 动作                    |
| --- | --- |
| 浮盈 > 10% + 趋势确认         | 可加仓                  |
| 每次加仓不超过原仓位         | 50%                     |
| 单只总仓位上限               | 30%                     |
| 趋势破坏（MA 多头转空头）    | 停止加仓，考虑减仓      |

> **加仓计划示例**：
> - 建仓 5%（首次买入）
> - 浮盈 10% + MA5 > MA20 → 加仓 2.5%（累计 7.5%）
> - 浮盈 20% + 成交量配合 → 再加仓 2.5%（累计 10%）
> - 浮盈 30% + 无需停止信号 → 再加仓 2.5%（累计 12.5%，接近 15% 上限）

## 决策流程与 Skill 协作

### 标准决策流程

```
研究标的 → 基本面筛选(ROE/增速/毛利)
         → 估值评估(PE/PEG)
         → 技术面确认(支撑/趋势)
         → 板块分析(轮动/风格)
         → 专家讨论(多空辩论)
         → 风险收益比计算
         → 仓位决策(凯利公式)
         → 建仓节奏(分批)
         → 持续跟踪(止损/止盈)
```

### Skill 协作流程

| 场景           | 推荐链路                                                               |
| -------------- | ---------------------------------------------------------------------- |
| 自上而下找机会 | `market` → `sector` → `screener` → `stock` → `technical` → `portfolio` |
| 已有个股做验证 | `stock` → `financial-analyst` → `sector` → `technical` → `portfolio`   |
| 持仓再平衡     | `portfolio` → `market` → `technical` → `screener` → `stock`            |
| 深度研究报告   | `investment-researcher` 总控，按需调用其他 skill                       |

交接时至少保留：市场状态、板块观点、候选池、基本面评级、技术触发、仓位计划、置信度。

## 关键经验（8 条铁律）

1. 不追高：PE>100 时风险极大
2. 板块轮动极快：不追轮动，持有核心仓位
3. 关键支撑位需多次测试确认，不赌单次
4. 仓位管理比选股重要
5. 现金是最好的期权：震荡市中 30% 现金是优势
6. 高赔率≠无风险：仍需止损纪律
7. 防御仓位（黄金/低估值金融）是组合压舱石
8. 科技仓位不能为零，至少 5-8%
