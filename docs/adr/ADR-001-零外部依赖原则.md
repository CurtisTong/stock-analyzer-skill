# ADR-001: 零外部依赖原则

| 字段     | 值                  |
| -------- | ------------------- |
| 状态     | Accepted            |
| 日期     | 2026-01-15          |
| 决策者   | curtis              |
| 相关模块 | `scripts/common/` 全模块 |

## 背景

stock-analyzer-skill 作为 Claude Code skill 分发，运行在用户本地环境。
Python 生态的数据分析库（pandas/numpy/requests）版本碎片化严重，
用户环境可能缺少 C 扩展或版本不兼容，导致 `pip install` 失败。

## 决策

**核心模块零硬依赖**：除 Python 标准库外，不强制要求任何第三方包。

- `requests`：可选依赖，缺失时降级为 `http.client`（stdlib）
- `akshare` / `pytdx` / `tushare`：可选数据源，缺失时跳过该源
- `yaml`（PyYAML）：**唯一例外**，作为构建时依赖（配置文件解析）

## 理由

1. **分发可靠性**：skill 安装后开箱即用，无需 `pip install`
2. **环境兼容**：覆盖 macOS / Linux / Windows + 各 Python 版本
3. **降级优雅**：可选依赖缺失时功能缩减而非崩溃

## 后果

- 正面：安装零摩擦，CI 矩阵简化
- 负面：部分功能需手写 stdlib 实现（如 HTTP 连接池、GBK 解码），代码量增加
- 负面：性能略低于专用库（pandas 向量化操作），但单股分析场景可接受
