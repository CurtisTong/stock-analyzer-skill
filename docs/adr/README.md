## ADR 规范

每个 ADR 文件命名：`ADR-XXX-标题.md`

内容结构：
- **背景**：问题陈述与上下文
- **决策**：选择的方案
- **理由**：为什么选择这个方案
- **后果**：正面和负面影响
- **相关**：关联的 ADR 或 TODO

## 新增 ADR

新增重大架构决策时创建新 ADR，编号递增。
