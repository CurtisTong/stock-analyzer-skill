## Phase A: 阻塞问题修复（P0 - 立即执行）

> 影响系统稳定性或数据正确性的问题，必须在任何功能开发前修复。

### A2. err() 函数改为异常抛出

| --- | --- |

| **时间估算** | 1.5h |
| **验收标准** | - [ ] err() 不再调用 sys.exit<br>- [ ] 批量处理单只失败不影响其他<br>- [ ] 错误信息仍然正确输出到 stderr |
