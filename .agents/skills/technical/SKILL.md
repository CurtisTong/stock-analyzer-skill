# Technical Analysis

纯技术分析报告——均线+指标+形态+趋势+A股特化+综合评分。不做基本面，不聊估值。

## Usage

```text
/technical <代码>              # 完整技术分析报告
/technical <代码> quick        # 快速技术面扫描（仅关键信号）
```

也可以自然语言触发："技术面分析 sh600989"、"纯技术角度看 sz000807"等。

## Instructions

使用中文。先给结论（综合评分+多空方向），再给逐项证据。涉及最新行情时必须运行脚本。

### Step 1: 运行技术分析脚本

```bash
cd ../../..
python3 scripts/technical.py sh600989               # 完整报告（日K 250根）
python3 scripts/technical.py sh600989 --quick        # 快速摘要
python3 scripts/technical.py sh600989 --scale 60     # 60分钟K线
python3 scripts/technical.py sh600989 -j             # JSON 输出
```

脚本自动完成数据获取（K线+实时行情）、所有指标计算、形态识别和信号汇总。

### Step 4: 解读要点

不同指标侧重不同：

- **均线系统**：中长期趋势方向。多头排列=强势，空头=弱势，粘合=即将变盘
- **MACD**：趋势动量。金叉+红柱放大=加速上涨，死叉+绿柱放大=加速下跌。背离是强烈反转信号
- **KDJ**：短线超买超卖。J>100=极度超买，J<0=极度超卖。注意：单边趋势中 KDJ 会钝化，此时金叉/死叉信号不可靠
- **BOLL**：波动率。带宽收窄=变盘前兆，价格触轨=极端位置
- **成交量**：量价配合=健康，量价背离=预警
- **K线形态**：短期转折信号，需与趋势方向结合判断
- **RSI**：辅助超买超卖，30/70 为常规分界线
- **涨跌停分析**：封涨停时技术指标暂停参考，需等次日开盘验证

## Guardrails

- 纯技术视角，不涉及基本面分析、估值分析、行业比较
- 涨跌停封板时明确提示"技术指标暂停参考"，不强行解读
- 连板股不给出"追板"建议，仅描述状态
- 数据时间戳必须在报告中可见
- 所有技术信号均标注市场条件限制（缩量下的金叉可靠性低于放量金叉等）
- KDJ 钝化时降低超买超卖信号的权重
- 不预测涨跌幅，不给目标价，仅给支撑/阻力位和概率倾向
- 明确声明不构成投资建议
