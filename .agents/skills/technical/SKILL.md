# Technical Analysis

纯技术分析报告——均线+指标+形态+趋势+A股特化+综合评分。不做基本面，不聊估值。

## Usage

```text
/technical <代码>                    # 完整技术分析报告
/technical <代码> quick              # 快速技术面扫描（仅关键信号）
/technical <代码> full               # 完整报告+个股分类+缠论+战法+市场自适应
```

也可以自然语言触发："技术面分析 sh600989"、"纯技术角度看 sz000807"等。

### Step 4: 解读要点

不同指标侧重不同：

- **均线系统**：中长期趋势方向。多头排列=强势，空头=弱势，粘合=即将变盘
- **MACD**：趋势动量。金叉+红柱放大=加速上涨，死叉+绿柱放大=加速下跌。背离是强烈反转信号
- **KDJ**：短线超买超卖。J>100=极度超买，J<0=极度超卖。注意：单边趋势中 KDJ 会钝化，此时金叉/死叉信号不可靠
- **BOLL**：波动率。带宽收窄=变盘前兆，价格触轨=极端位置
- **成交量**：量价配合=健康，量价背离=预警
- **K线形态**：短期转折信号，需与趋势方向结合判断
- **RSI**：辅助超买超卖，30/70 为常规分界线
- **涨跌停分析**：封涨停时技术指标暂停参考，需等次日开盘验证

### Step 5: 缠论与本土战法解读（--classify 模式）

- **缠论中枢**：3段重叠区间构成中枢，价格相对中枢位置决定多空格局
- **一买**：离开中枢的底背驰结束点，熊市/冰点中加权；**二买**：不创新低的次低点；**三买**：突破中枢后回踩不落入，牛市加权
- **背驰**：价格创新高/低但 MACD 面积缩小，预示趋势反转
- **三阴一阳**：底部洗盘结束信号，阳线需放量覆盖前阴线
- **老鸭头**：鸭颈→鸭头→鸭嘴三阶段，鸭嘴确认时放量突破前高为可靠信号
- **美人肩**：强势上升后横盘不破 MA10，缩量后再放量突破
- **涨停双响炮**：首板+缩量整理+再封板放量，确认强势二次攻击

### Step 6: 个股类型 × 指标权重

综合评分会根据个股类型自动调整各指标权重：

| 类型     | 加权指标                            | 降权指标          |
| -------- | ----------------------------------- | ----------------- |
| 题材股   | K线形态×1.5, 涨停分析×1.5, 量比×1.3 | MACD×0.5, KDJ×0.5 |
| 强成长股 | MACD×1.3, BOLL×1.2, 量比×1.2        | KDJ×0.4           |
| 周期股   | MACD×1.3, KDJ×1.2, 缠论×1.3         | 均线×0.6          |
| 蓝筹股   | 均线×1.3, BOLL×1.2, 箱体×1.1        | KDJ×0.4           |

## Guardrails

- 纯技术视角，不涉及基本面分析、估值分析、行业比较
- 涨跌停封板时明确提示"技术指标暂停参考"，不强行解读
- 连板股不给出"追板"建议，仅描述状态
- 数据时间戳必须在报告中可见
- 所有技术信号均标注市场条件限制（缩量下的金叉可靠性低于放量金叉等）
- KDJ 钝化时降低超买超卖信号的权重
- 缠论买卖点仅为技术参考，需结合市场环境判断
- 不预测涨跌幅，不给目标价，仅给支撑/阻力位和概率倾向
- 明确声明不构成投资建议
